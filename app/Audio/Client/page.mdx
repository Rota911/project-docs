# Audio Client

Az Audio modul client oldali funkciói.

## Funkciók

### PlayAudio

Hangot játszik le a frontend-ből (csak a player hallja).

```lua
local soundId = Audio:PlayAudio(soundName)
```

**Paraméterek:**
- `soundName` (string) - A lejátszandó hang neve (pl. `"beep_sp"`)

**Visszatérési érték:**
- `soundId` (number) - A hang egyedi azonosítója, amivel később leállítható

**Példa:**
```lua
local Audio = LS:ImportModule('Audio')

-- Sípoló hang lejátszása
local soundId = Audio:PlayAudio('beep_sp')

-- Telefon csengőhang
local phoneSound = Audio:PlayAudio('phone_ring0_sp')
```

---

### PlaySoundFromCoord

Hangot játszik le egy megadott koordinátáról. Mindenki hallja, aki közel van.

```lua
local soundId = Audio:PlaySoundFromCoord(soundName, coords)
```

**Paraméterek:**
- `soundName` (string) - A lejátszandó hang neve
- `coords` (vector3 | vector4) - A koordináták, ahonnan a hang jön

**Visszatérési érték:**
- `soundId` (number) - A hang egyedi azonosítója

**Példa:**
```lua
local Audio = LS:ImportModule('Audio')

-- Ajtó kinyitás hang egy pozícióról
local doorCoords = vector3(123.45, 456.78, 28.5)
local soundId = Audio:PlaySoundFromCoord('unlock_door_sp', doorCoords)

-- Csengő hang az aktuális pozícióról
local myPos = GetEntityCoords(PlayerPedId())
Audio:PlaySoundFromCoord('doorbell_sp', myPos)
```

---

### PlaySoundFromEntity

Hangot játszik le egy entitásról (jármű, ped, objektum). A hang követi az entitást.

```lua
local soundId = Audio:PlaySoundFromEntity(soundName, entity)
```

**Paraméteters:**
- `soundName` (string) - A lejátszandó hang neve
- `entity` (number) - Az entitás handle

**Visszatérési érték:**
- `soundId` (number) - A hang egyedi azonosítója

**Példa:**
```lua
local Audio = LS:ImportModule('Audio')

-- Bilincs hang a playeren
local playerPed = PlayerPedId()
Audio:PlaySoundFromEntity('metal_cuff_on_sp', playerPed)

-- Motor hang járművön
local vehicle = GetVehiclePedIsIn(PlayerPedId(), false)
if vehicle ~= 0 then
    Audio:PlaySoundFromEntity('anchor_drop_sp', vehicle)
end
```

---

### StopAudio

Leállít egy lejátszás alatt lévő hangot.

```lua
local success = Audio:StopAudio(soundId)
```

**Paraméterek:**
- `soundId` (number) - A leállítandó hang azonosítója (amit a PlayAudio/PlaySoundFromCoord/PlaySoundFromEntity adott vissza)

**Visszatérési érték:**
- `success` (boolean) - `true` ha sikerült leállítani, `false` ha nem létezik a soundId

**Példa:**
```lua
local Audio = LS:ImportModule('Audio')

-- Hang lejátszása
local soundId = Audio:PlayAudio('phone_ring0_sp')

-- 5 másodperc után leállítás
Wait(5000)
local stopped = Audio:StopAudio(soundId)

if stopped then
    print('Hang sikeresen leállítva')
else
    print('Hang nem található vagy már befejeződött')
end
```

## Automatikus Hangkezelés

A modul automatikusan figyeli a lejátszott hangokat és felszabadítja azokat, amelyek befejeződtek. Nem szükséges manuálisan minden hangot leállítani, csak akkor, ha idő előtt szeretnéd befejezni.

## Példa Használatok

### Telefon Csengés

```lua
local Audio = LS:ImportModule('Audio')

-- Telefon csengés indítása
local ringingSound = Audio:PlayAudio('phone_ring1_sp')

-- Felvétel után leállítás
RegisterCommand('pickup', function()
    Audio:StopAudio(ringingSound)
    Audio:PlayAudio('calling_sp')
end)
```

### Zárfeltörés Hang Effekt

```lua
local Audio = LS:ImportModule('Audio')

function PlayLockpickSounds()
    CreateThread(function()
        for i = 1, 4 do
            Audio:PlayAudio('pick' .. i .. '_sp')
            Wait(500)
        end
    end)
end

-- Használat
PlayLockpickSounds()
```

### Pozíció Alapú Esemény Hang

```lua
local Audio = LS:ImportModule('Audio')

-- Esemény fogadása serverről
RegisterNetEvent('myresource:playDoorbell', function(coords)
    Audio:PlaySoundFromCoord('doorbell_sp', coords)
end)
```

### Jármű Hang Effekt

```lua
local Audio = LS:ImportModule('Audio')

RegisterCommand('anchor', function()
    local ped = PlayerPedId()
    local vehicle = GetVehiclePedIsIn(ped, false)
    
    if vehicle ~= 0 then
        Audio:PlaySoundFromEntity('anchor_drop_sp', vehicle)
    end
end)
```

## Megjegyzések

- A hangok betöltése automatikusan megtörténik első lejátszáskor
- Ha a hang nem töltődik be 5 másodpercen belül, a funkció `nil` értékkel tér vissza
- Az entitás alapú hangok automatikusan követik az entitást
- Debug logokat a Logger modul Debug szintjén lehet látni
