# ls_nativeaudio

Az **ls_nativeaudio** egy custom audio rendszer natív GTA és saját custom hangok betöltésére és lejátszására.

## Áttekintés

Custom AWC hangfájlok betöltése és lejátszása a játékban. \

## Fontos tudnivalók
1db audio dir maximum 30 hangot tud tárolni. \
Scriptenként maximum 4db audio dir lehet. \
Hangok lejátszása syncelve van a FiveM által így nem kell feleslegesen 6x mindenkinél lejátszani a hangot. \
A hangok listáját általában egy json fájlban meg tudod tekinteni amit az AudioTool config-ja! 

## Használat

Minden hang visszaadja a soundId-t amivel letudjátok állítani a hangot. \
Fontos tudnivaló ha leáll a hang a soundID 1000% hogy a Rage és a Garbage Collector release-eli az id-t!  \
Ha release-elt a soundId akkor **NE** használd a soundId-t!

# PlaySound
Ezzel kizárolag a játékos hallja a hangot.
Ez arra jó ha valami UI hangot akarsz elindítani nála.

```lua
 local soundId = exports.ls_nativeaudio:PlaySound("sound_name", "sound_bank", "sound_dict")
 ```
# PlaySoundFromCoord
Ezzel a megadott koordinátán hangot indít.

```lua
---@field Coords Vector3
 local soundId = exports.ls_nativeaudio:PlaySoundFromCoord("sound_name", "sound_bank", "sound_dict", Coords)
 ```

# PlaySoundFromPed
Ezzel a játékos pedjén indít hangot.

```lua
 local soundId = exports.ls_nativeaudio:PlaySoundFromPed("sound_name", "sound_bank", "sound_dict")
 ```
 
 # StopSound
Ezzel megállítod a hangot.

```lua
---@field soundId number
exports.ls_nativeaudio:StopSound(soundId)
 ```
