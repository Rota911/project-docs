# ls_nativeaudio

Az **ls_nativeaudio** egy custom audio rendszer natív GTA hangok betöltésére.

## Áttekintés

Custom AWC hangfájlok betöltése és lejátszása a játékban.

### Főbb funkciók

- 🔊 **Custom hangok** - AWC formátum support
- 🎵 **Sound banks** - Hang csoportok kezelése
- 📍 **3D audio** - Pozíció alapú hangok
- 🎚️ **Hangerő vezérlés** - Volume és pitch beállítás
- 🔄 **Loop support** - Ismétlődő hangok

## Telepítés

### Függőségek

Nincs függőség, standalone resource.

### server.cfg

```cfg
ensure ls_nativeaudio
```

## Hang fájlok

### Struktúra

```
ls_nativeaudio/
├── audiodirectory/
│   ├── basic.awc          # Hang fájl
│   └── basic.awc.xml      # Metadata
├── data/
│   └── generalData1.dat54.rel  # Sound bank definíció
└── fxmanifest.lua
```

### AWC fájl létrehozása

1. **Audio fájlok készítése** (WAV, MP3)
2. **Konvertálás AWC-re** - Audiotool használata
3. **XML metadata** - Hang nevekkel és beállításokkal
4. **DAT54 fájl** - Sound bank definíció

## Használat

### Hang lejátszása

```lua
-- Client side
-- Egyszerű hang
exports.ls_nativeaudio:PlaySound('sound_name', 'sound_bank')

-- Példa: Button kattintás
exports.ls_nativeaudio:PlaySound('button_click', 'basic')

-- Hangerő beállítással
exports.ls_nativeaudio:PlaySoundWithVolume('sound_name', 'sound_bank', 0.5)

-- Loop hang
exports.ls_nativeaudio:PlayLoopedSound('loop_sound', 'sound_bank')
```

### 3D pozícionált hang

```lua
-- Hang lejátszása koordinátán
local soundId = exports.ls_nativeaudio:PlaySoundAt(
    'alarm',                              -- Hang név
    'basic',                              -- Sound bank
    vector3(100.0, 200.0, 30.0),         -- Koordináták
    50.0                                  -- Max távolság
)

-- Hang leállítása
exports.ls_nativeaudio:StopSound(soundId)
```

### Entity-hez kapcsolt hang

```lua
-- Vehicle-hez hang
local vehicle = GetVehiclePedIsIn(PlayerPedId(), false)
local soundId = exports.ls_nativeaudio:PlaySoundOnEntity(
    'engine_sound',
    'vehicle',
    vehicle
)

-- Ped-hez hang
local ped = PlayerPedId()
exports.ls_nativeaudio:PlaySoundOnEntity('footstep', 'basic', ped)
```

### Hang leállítása

```lua
-- Konkrét hang
exports.ls_nativeaudio:StopSound(soundId)

-- Összes hang
exports.ls_nativeaudio:StopAllSounds()

-- Konkrét bank összes hangja
exports.ls_nativeaudio:StopSoundsFromBank('basic')
```

## Audio Tool használata

### Konvertálás

```bash
# WAV → AWC konvertálás
cd tool/
./Audiotool.exe convert input.wav output.awc

# Metadata generálás
./Audiotool.exe metadata output.awc
```

### AWC XML struktúra

```xml
<?xml version="1.0" encoding="UTF-8"?>
<AudioWaveContainer>
    <Version value="1" />
    <ChunkIndices capacity="1">
        <Item>
            <Name>button_click</Name>
            <Offset>0</Offset>
            <Size>12345</Size>
        </Item>
    </ChunkIndices>
</AudioWaveContainer>
```

### DAT54 REL

```xml
<?xml version="1.0" encoding="UTF-8"?>
<Dat54>
    <Items>
        <Item type="SimpleSound">
            <Name>button_click</Name>
            <Header>
                <Category>PLAYER</Category>
                <Volume>0.8</Volume>
            </Header>
            <ContainerName>basic</ContainerName>
            <FileName>button_click</FileName>
        </Item>
    </Items>
</Dat54>
```

## API Export-ok

### Client

```lua
-- Alap lejátszás
exports.ls_nativeaudio:PlaySound(soundName, bankName)

-- Hangerővel
exports.ls_nativeaudio:PlaySoundWithVolume(soundName, bankName, volume)

-- 3D hang
exports.ls_nativeaudio:PlaySoundAt(soundName, bankName, coords, maxDistance)

-- Entity hang
exports.ls_nativeaudio:PlaySoundOnEntity(soundName, bankName, entity)

-- Loop hang
exports.ls_nativeaudio:PlayLoopedSound(soundName, bankName)

-- Hang leállítás
exports.ls_nativeaudio:StopSound(soundId)
exports.ls_nativeaudio:StopAllSounds()
exports.ls_nativeaudio:StopSoundsFromBank(bankName)

-- Bank betöltés/eltávolítás
exports.ls_nativeaudio:LoadBank(bankName)
exports.ls_nativeaudio:UnloadBank(bankName)
```

## Gyakorlati példák

### Button hangok

```lua
-- UI button kattintás
RegisterNUICallback('buttonClick', function()
    exports.ls_nativeaudio:PlaySound('button_click', 'basic')
end)

-- Success hang
AddEventHandler('notification:success', function()
    exports.ls_nativeaudio:PlaySound('success', 'basic')
end)

-- Error hang
AddEventHandler('notification:error', function()
    exports.ls_nativeaudio:PlaySound('error', 'basic')
end)
```

### Vehicle hangok

```lua
-- Engine start hang
RegisterNetEvent('vehicle:engineStart')
AddEventHandler('vehicle:engineStart', function(vehicle)
    exports.ls_nativeaudio:PlaySoundOnEntity('engine_start', 'vehicle', vehicle)
end)

-- Car lock hang
RegisterNetEvent('vehicle:lock')
AddEventHandler('vehicle:lock', function(vehicle)
    exports.ls_nativeaudio:PlaySoundOnEntity('car_lock', 'vehicle', vehicle)
end)

-- Car alarm
RegisterNetEvent('vehicle:alarm')
AddEventHandler('vehicle:alarm', function(vehicle)
    local soundId = exports.ls_nativeaudio:PlayLoopedSoundOnEntity('car_alarm', 'vehicle', vehicle)
    
    -- 30 másodperc után leállítás
    SetTimeout(30000, function()
        exports.ls_nativeaudio:StopSound(soundId)
    end)
end)
```

### Lockpick rendszer

```lua
RegisterNetEvent('lockpick:start')
AddEventHandler('lockpick:start', function(doorCoords)
    -- Lockpick hang loop
    local soundId = exports.ls_nativeaudio:PlaySoundAt(
        'lockpick',
        'basic',
        doorCoords,
        10.0  -- 10 méter távolságig hallható
    )
    
    -- Progress bar
    LS.UI:Progress({
        duration = 10000,
        label = 'Zár feltörése...'
    }, function(cancelled)
        exports.ls_nativeaudio:StopSound(soundId)
        
        if not cancelled then
            -- Success hang
            exports.ls_nativeaudio:PlaySound('lockpick_success', 'basic')
        end
    end)
end)
```

### Metal detector

```lua
local detecting = false
local detectSoundId = nil

RegisterNetEvent('metaldetector:toggle')
AddEventHandler('metaldetector:toggle', function()
    detecting = not detecting
    
    if detecting then
        -- Detector bekapcs hang
        exports.ls_nativeaudio:PlaySound('detector_on', 'basic')
        
        -- Loop beep hang
        detectSoundId = exports.ls_nativeaudio:PlayLoopedSound('detector_beep', 'basic')
    else
        -- Detector kikapcs
        exports.ls_nativeaudio:PlaySound('detector_off', 'basic')
        
        if detectSoundId then
            exports.ls_nativeaudio:StopSound(detectSoundId)
            detectSoundId = nil
        end
    end
end)
```

### Weather hangok

```lua
-- Eső hang
Citizen.CreateThread(function()
    local rainSoundId = nil
    
    while true do
        Wait(5000)
        
        local weather = GetPrevWeatherTypeHashName()
        
        if weather == `RAIN` or weather == `THUNDER` then
            if not rainSoundId then
                rainSoundId = exports.ls_nativeaudio:PlayLoopedSound('rain_ambient', 'weather')
            end
        else
            if rainSoundId then
                exports.ls_nativeaudio:StopSound(rainSoundId)
                rainSoundId = nil
            end
        end
    end
end)
```

### Cuff hangok

```lua
RegisterNetEvent('police:cuff')
AddEventHandler('police:cuff', function()
    local ped = PlayerPedId()
    
    -- Bilincs hang
    exports.ls_nativeaudio:PlaySound('cuff', 'basic')

        duration = -1

AddEventHandler('police:uncuff', function()
    -- Bilincs levétel hang
    exports.ls_nativeaudio:PlaySound('uncuff', 'basic')

end)
```

## Custom sound bank létrehozás

### 1. Hangok gyűjtése

```
sounds/
├── button_click.wav
├── success.wav
├── error.wav
└── notification.wav
```

### 2. Konvertálás

```bash
Audiotool.exe convert button_click.wav button_click.awc
# ... többi fájl
```

### 3. AWC összevonás

```bash
Audiotool.exe pack basic.awc button_click.awc success.awc error.awc notification.awc
```

### 4. XML metadata

```xml
<AudioWaveContainer>
    <ChunkIndices>
        <Item><Name>button_click</Name></Item>
        <Item><Name>success</Name></Item>
        <Item><Name>error</Name></Item>
        <Item><Name>notification</Name></Item>
    </ChunkIndices>
</AudioWaveContainer>
```

### 5. DAT54 REL

```xml
<Dat54>
    <Items>
        <Item type="SimpleSound">
            <Name>button_click</Name>
            <Header><Category>UI</Category></Header>
            <ContainerName>basic</ContainerName>
            <FileName>button_click</FileName>
        </Item>
        <!-- ... többi hang -->
    </Items>
</Dat54>
```

### 6. fxmanifest.lua

```lua
fx_version 'cerulean'
game 'gta5'

files {
    'audiodirectory/basic.awc',
    'audiodirectory/basic.awc.xml',
    'data/generalData1.dat54.rel'
}

data_file 'AUDIO_WAVEPACK' 'audiodirectory'
data_file 'AUDIO_SOUNDDATA' 'data/generalData1.dat54.rel'
```

## Troubleshooting

### Hang nem játszódik le

```lua
-- Ellenőrizd a bank betöltését
exports.ls_nativeaudio:LoadBank('basic')

-- Várj a betöltésre
Wait(1000)

-- Próbáld újra
exports.ls_nativeaudio:PlaySound('sound_name', 'basic')
```

### Túl halk a hang

```lua
-- Hangerő növelése
exports.ls_nativeaudio:PlaySoundWithVolume('sound_name', 'basic', 1.0)

-- Vagy XML-ben:
<Volume>1.0</Volume>  <!-- 0.0 - 1.0 -->
```
