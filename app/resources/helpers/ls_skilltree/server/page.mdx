# Server

## ST Module

### ST:Create

```lua
---@param data SkillTree
---@return SkillTreeInstance|nil
local tree = ST:Create({
    id = "combat",
    name = "Combat",
    nodes = {
        ["node1"] = {
            preRequsites = {},
            children = { "node2" },
            lock = {},
            unlocked = false,
            label = "Basic Combat",
            description = "Learn basic combat skills",
            cost = 1
        },
        ["node2"] = {
            preRequsites = { "node1" },
            children = {},
            lock = {},
            unlocked = false,
            label = "Advanced Combat",
            description = "Learn advanced combat skills",
            cost = 2
        }
    },
    xpGrowth = 1.15,
    baseXp = 500
})
```

### tree:UnlockNode

```lua
---@param charId number
---@param nodeId string
---@return boolean success
---@return string|nil reason
local success, reason = tree:UnlockNode(charId, "node1")
```

| Reason | Description |
|---|---|
| `invalid_node` | Node does not exist in the tree |
| `missing_prerequisite` | A prerequisite node is not unlocked |
| `not_enough_points` | Character does not have enough skill points |
| `locked_by_other_node` | Another unlocked node blocks this one |
