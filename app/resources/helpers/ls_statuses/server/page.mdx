# Server

## StatusHandler

```lua
local StatusHandler = require 'server.handlers.statusHandler'
```

### Register

```lua
---@param name string
---@param statusData StatusData
---@return boolean
StatusHandler:Register('radiation', {
    baseValue = 0,
    icon = 'IconRadioactiveFilled',
    color = 'lime',
    flash = true,
    modify = 0.5,
    options = {
        hide = true,
        inverted = true,
        order = 8,
        protected = true
    }
})
```

### Get

```lua
---@param status? string
---@return table
local allStatuses = StatusHandler:Get()
local singleStatus = StatusHandler:Get('hunger')
```

### Modify:Add

```lua
---@param identifier number|string
---@param status string
---@param value number
---@return boolean
StatusHandler.Modify:Add(source, 'hunger', 25)
```

### Modify:Remove

```lua
---@param identifier number|string
---@param status string
---@param value number
---@return boolean
StatusHandler.Modify:Remove(source, 'hunger', 10)
```

### Modify:Set

```lua
---@param identifier number|string
---@param status string
---@param value number
---@return boolean
StatusHandler.Modify:Set(source, 'hunger', 100)
```

## Types

### StatusData

```lua
---@class StatusData
---@field baseValue number|boolean
---@field icon string
---@field color string
---@field flash boolean
---@field modify? number
---@field options? StatusOptions
```

### StatusOptions

```lua
---@class StatusOptions
---@field noReset? boolean
---@field inverted? boolean
---@field hide? boolean
---@field order? number
---@field protected? boolean
```
