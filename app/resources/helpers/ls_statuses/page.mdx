# ls_statuses

Karakter státusz rendszer (éhség, szomjúság, stressz, oxigén, stb.) automatikus decay-jel és szerver-kliens szinkronizációval.

## Áttekintés

Az `ls_statuses` egy percenkénti cron-alapú decay rendszert használ szerver oldalon, miközben a kliens interpolálja az értékeket a frissítések között a sima vizuális megjelenítéshez.

## Függőségek

- `ls_core`

## Státusz definíció

### StatusData

```lua
---@class StatusData
---@field baseValue number|boolean Alapérték (0-100 vagy boolean)
---@field icon string Tabler ikon neve
---@field color string Szín (Mantine szín)
---@field flash boolean Villogjon-e alacsony értéknél
---@field modify? number Decay mérték (%/perc)
---@field options? StatusOptions Opcionális beállítások
```

### StatusOptions

```lua
---@class StatusOptions
---@field noReset? boolean Ne állítsa vissza alapértékre reset-nél
---@field inverted? boolean 0%-tól 100%-ig megy (pl. stressz, részegség)
---@field hide? boolean Rejtse el ha az érték 0 vagy 100
---@field order? number Megjelenítési sorrend
---@field protected? boolean Kliens oldali módosítás tiltása
```

| `thirst` | 100 | 1.66%/perc | indigo | ✗ |
| `hunger` | 100 | 1.66%/perc | yellow | ✗ |
| `stress` | 0 | 1.66%/perc | red | ✓ |
| `drunk` | 0 | 3.33%/perc | pink | ✓ |
| `oxygen-tank` | 0 | — | white | ✓ |

### Ideiglenes státuszok (`sh_tempStatuses.lua`)

| Státusz | Alapérték | Decay | Leírás |
|---------|-----------|-------|--------|
| `oxygen` | 100 | — | Oxigén szint |
| `stamina` | 100 | — | Stamina szint |
| `heal_ticks` | 0 | 50%/perc | HP regeneráció (szuri hatás) |
| `speed_ticks` | 0 | 50%/perc | Sebesség boost (adrenalin) |
| `pursuit-modes` | 3 | — | Járőr mód (1-5) |

## Server API

### StatusHandler

```lua
local StatusHandler = require 'server.handlers.statusHandler'
```

### Register

Új státusz regisztrálása.

```lua
---@param name string Státusz azonosító
---@param statusData StatusData Státusz adatok
---@return boolean
StatusHandler:Register(name, statusData)
```

### Get

Státusz(ok) lekérése.

```lua
---@param status? string Státusz neve (nil = összes)
---@return table
StatusHandler:Get(status)
```

### Modify:Add

Érték hozzáadása a karakter státuszához.

```lua
---@param identifier number|string Karakter ID vagy source
---@param status string Státusz neve
---@param value number Hozzáadandó érték
---@return boolean
StatusHandler.Modify:Add(identifier, status, value)
```

### Modify:Remove

Érték elvétele a karakter státuszától.

```lua
---@param identifier number|string Karakter ID vagy source
---@param status string Státusz neve
---@param value number Elvett érték
---@return boolean
StatusHandler.Modify:Remove(identifier, status, value)
```

### Modify:Set

Konkrét érték beállítása.

```lua
---@param identifier number|string Karakter ID vagy source
---@param status string Státusz neve
---@param value number Új érték (0-100)
---@return boolean
StatusHandler.Modify:Set(identifier, status, value)
```

## Client oldal

### Státusz frissítés event

A kliens másodpercenként interpolálja a státusz értékeket a szerver frissítések között.

```lua
AddEventHandler('Statuses:Client:StatusUpdate', function(statuses)
    -- statuses = { hunger = 85.3, thirst = 72.1, stress = 12.5, ... }
end)
```

### Szerver frissítés event

A kliens `Core:Client:CharacterDataChanged` event-en keresztül kapja az új szerver értékeket:

```lua
RegisterNetEvent('Core:Client:CharacterDataChanged', function(key, value)
    if key == 'statuses' then
        -- value = { hunger = 85.0, thirst = 72.0, ... }
    end
end)
```

## Decay rendszer

A szerver `LS.Cron`-t használ percenkénti decay futtatáshoz:

1. Minden percben végigiterál az összes online karakteren
2. Kiszámítja az eltelt időt az utolsó frissítés óta
3. A `modify` értéket alkalmazza: `newVal = currentVal - (modify * elapsedMinutes)`
4. Az értéket 0 és 100 közé szorítja
5. Frissíti a karakter adatait `character:SetData('statuses', ...)`

## Példák

### Étel fogyasztás

```lua
local StatusHandler = require 'server.handlers.statusHandler'

LS.Callback:Register('Food:Eat', function(source, itemData)
    local success = StatusHandler.Modify:Add(source, 'hunger', itemData.restoreAmount)
    if success then
        LS.UI:Notification(source, {
            type = 'success',
            title = 'Étkezés',
            message = ('Éhség feltöltve +%d%%'):format(itemData.restoreAmount)
        })
    end
    return success
end)
```

### Egyedi státusz regisztrálása

```lua
local StatusHandler = require 'server.handlers.statusHandler'

StatusHandler:Register('radiation', {
    baseValue = 0,
    icon = 'IconRadioactiveFilled',
    color = 'lime',
    flash = true,
    modify = 0.5,
    options = {
        hide = true,
        inverted = true,
        order = 8,
        protected = true
    }
})
```

### Státusz lekérdezése

```lua
local character = GetCharacter(source)
local statuses = character:GetData('statuses') or {}
local hunger = statuses.hunger or 100

if hunger < 10 then
    StatusHandler.Modify:Add(source, 'stress', 5)
end
```
