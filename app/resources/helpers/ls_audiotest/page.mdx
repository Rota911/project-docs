# ls_audiotest

Az **ls_audiotest** egy egyszerű audio teszt eszköz fejlesztéshez.

## Áttekintés

Gyors audio fájl tesztelés a játékban native audio rendszerrel.

### Főbb funkciók

- 🎵 **Gyors teszt** - Audio lejátszás parancsból
- 📊 **Debug info** - Hang információk megjelenítése
- 🔊 **Sound bank teszt** - Bank betöltés ellenőrzés

## Telepítés

### Függőségek

```lua
dependencies {
    'ls_nativeaudio'
}
```

### server.cfg

```cfg
ensure ls_nativeaudio
ensure ls_audiotest
```

## Használat

### Parancsok

```lua
-- Hang lejátszás
/audiotest play [soundName] [bankName]

-- Példa:
/audiotest play button_click basic

-- Bank lista
/audiotest banks

-- Sound lista egy bankban
/audiotest sounds [bankName]

-- Hang leállítás
/audiotest stop

-- Összes hang leállítás
/audiotest stopall
```

### Példák

```lua
-- UI hang teszt
/audiotest play success basic

-- Vehicle hang
/audiotest play engine_start vehicle

-- Loop hang
/audiotest play alarm basic

-- Leállítás
/audiotest stop
```

## Fejlesztés

### Custom audio teszt

```lua
-- client.lua
RegisterCommand('testmysound', function()
    -- Saját hang tesztelése
    exports.ls_nativeaudio:PlaySound('my_custom_sound', 'my_bank')
    
    print('Hang lejátszva: my_custom_sound')
end)
```

### Debug logging

```lua
-- Sound info kiírása
RegisterCommand('soundinfo', function(source, args)
    local soundName = args[1]
    local bankName = args[2]
    
    print('Sound:', soundName)
    print('Bank:', bankName)
    print('Testing...')
    
    local success = exports.ls_nativeaudio:PlaySound(soundName, bankName)
    
    if success then
        print('✓ Sikeres lejátszás')
    else
        print('✗ Hiba a lejátszásban')
    end
end)
```
