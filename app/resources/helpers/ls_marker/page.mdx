# ls_marker

Az **ls_marker** egy fejlett DUI-alapú marker rendszer dinamikus 3D markerek létrehozásához és kezeléséhez FiveM szervereken.

## Áttekintés

Modern marker rendszer amely DUI (Drawing User Interface) technológiát használ React alapú UI komponensekkel, lehetővé téve teljes testreszabhatóságot és valós idejű interakciót.

### Főbb funkciók

- 🎯 **DUI-alapú rendering** - React + TypeScript alapú vizuális komponensek
- 📍 **Mark markerek** - 3D nyíl és gyémánt markerek ikonokkal
- 🅿️ **Ground markerek** - Parkoló helyek jármű detektálással
- 🔷 **GroundPoly markerek** - Poligon alakú parkoló zónák
- 📌 **Waypoint markerek** - Távolság-alapú checkpoint markerek
- 🚗 **Kerék detektálás** - Precíz jármű pozíció ellenőrzés
- 🎨 **Valós idejű szín váltás** - Státusz alapú színek (zöld, narancssárga, fehér)
- 🔌 **Plugin rendszer** - Egyszerű integrációs rendszer más resource-okhoz
- 💾 **Export funkciók** - Teljes API más resource-ok számára


## Architektúra

### DUI Rendszer

Az ls_marker egy egyedi DUI (Drawing User Interface) megközelítést használ:

1. **React Frontend** (`src/` mappa)
   - Vite + React + TypeScript + Tailwind CSS stack
   - Komponensek: `DiamondMark.tsx`, `GroundAreaMark.tsx`, `GroundPolyMark.tsx`, `Marker.tsx`
   - Valós idejű frissítés NUI üzeneteken keresztül

2. **Lua Backend** (`client/` mappa)
   - DUI instance kezelés (`client/modules/dui.lua`)
   - Marker típusok class-alapú rendszerrel (`client/classes/`)
   - Központi marker manager (`client/modules/marker.lua`)

3. **Renderelési folyamat**
   - DUI létrehozása → Runtime texture generálás → 3D világba rajzolás
   - `DrawTexturedPoly` használata forgott textúrákhoz
   - Dinamikus távolság-alapú méretezés

### Marker típusok

```
TypeHandlers = {
  waypoint    → client/classes/waypoint.lua
  mark        → client/classes/mark.lua
  ground      → client/classes/ground.lua
  groundpoly  → client/classes/groundpoly.lua
}
```

Minden típus implementálja:
- `ResolveUrl()` - DUI URL meghatározás
- `OnDuiReady()` - Inicializálás üzenet küldés
- `CanRender()` - Render feltétel ellenőrzés
- `RenderSingle()` - Egyedi render logika
- `GetStreamDistance()` - Streaming távolság kalkuláció

  - **Fehér/Szürke**: Nincs jármű vagy távol van
  - **Narancssárga**: Részben bent (nem minden kerék)
  - **Zöld**: Teljesen bent (mind a 4+ kerék)
- Z-tolerancia: ±2 méter
- Csak a játékos járműve detektálódik
- 250ms frissítési ráta

### Példa: Mission System

```lua
-- client/mission.lua
local currentMission = {
  waypoints = {},
  currentWaypointIndex = 1
}

function StartDeliveryMission(locations)
  -- Markerek létrehozása az összes lokációhoz
  for i, loc in ipairs(locations) do
    local wp = exports.ls_marker:CreateWaypoint({
      id = ("mission_wp_%d"):format(i),
      position = loc.coords,
      color = i == 1 and "#4dabf7" or "#6c757d",  -- Első kék, többi szürke
      label = ("WAYPOINT %d/%d"):format(i, #locations),
      streamDistance = 300.0
    })
    
    table.insert(currentMission.waypoints, wp.id)
  end
  
  LS.UI:Notify('Kövesd a waypoint-okat!', 'info')
end

-- Checkpoint elérés kezelése
CreateThread(function()
  while true do
    Wait(500)
    
    if #currentMission.waypoints > 0 then
      local currentWpId = currentMission.waypoints[currentMission.currentWaypointIndex]
      local marker = exports.ls_marker:GetMarker(currentWpId)
      
      if marker then
        local playerPos = GetEntityCoords(cache.ped)
        local dist = #(playerPos - marker.position)
        
        if dist < 5.0 then
          -- Waypoint elérve
          exports.ls_marker:DeleteMarker(currentWpId)
          
          currentMission.currentWaypointIndex = currentMission.currentWaypointIndex + 1
          
          if currentMission.currentWaypointIndex <= #currentMission.waypoints then
            -- Következő waypoint aktiválása
            local nextWpId = currentMission.waypoints[currentMission.currentWaypointIndex]
            exports.ls_marker:UpdateMarkerVisual(nextWpId, "color", "#4dabf7")
            LS.UI:Notify(('Waypoint %d/%d'):format(
              currentMission.currentWaypointIndex, 
              #currentMission.waypoints
            ), 'success')
          else
            -- Mission befejezve
            LS.UI:Notify('Mission completed!', 'success')
            currentMission.waypoints = {}
            currentMission.currentWaypointIndex = 1
          end
        end
      end
    end
  end
end)
```

```lua
-- client/parking.lua
local ParkingSystem = {}
ParkingSystem.spots = {}

function ParkingSystem:CreateParkingLot(name, spots)
  for i, spot in ipairs(spots) do
    local markerId = ("%s_spot_%d"):format(name, i)
    
    local marker = exports.ls_marker:CreateGround({
      id = markerId,
      position = spot.position,
      heading = spot.heading,
      width = spot.width or 17.2,
      height = spot.height or 13.2,
      streamDistance = 75.0,
      
      label = ("PARKING #%d"):format(i),
      iconName = "IconParking",
      
      activeBorderColor = "#69db7c",
      errorBorderColor = "#ffa94d",
      
      onVehicleParked = function(mId, vehicle)
        -- Parkolás mentése
        local plate = GetVehicleNumberPlateText(vehicle)
        TriggerServerEvent('parking:saveVehicle', mId, plate, spot.position)
        
        -- Marker tiltása (foglalt)
        exports.ls_marker:DisableMarker(mId)
        ParkingSystem.spots[mId] = { vehicle = vehicle, plate = plate }
        
        LS.UI:Notify('Jármű leadva', 'success')
      end,
      
      onVehicleExited = function(mId, vehicle)
        -- Ha elhagyja a helyet mielőtt leadná
        LS.UI:Notify('Parkold le teljesen a járművet!', 'warning')
      end
    })
    
    self.spots[markerId] = { marker = marker, occupied = false }
  end
end

function ParkingSystem:RetrieveVehicle(markerId)
  local spot = self.spots[markerId]
  if spot and spot.vehicle then
    -- Jármű előhívása
    TriggerServerEvent('parking:retrieveVehicle', spot.plate)
    
    -- Marker újra engedélyezése
    exports.ls_marker:EnableMarker(markerId)
    spot.occupied = false
    spot.vehicle = nil
  end
end
```

```lua
        while #(GetEntityCoords(PlayerPedId()) - point) > 10.0 do
            Wait(1000)
        end
        
        -- Delivery akció
        LS.UI:Notify(('Delivery %d/%d kész'):format(i, #deliveryRoute), 'success')
        TriggerServerEvent('delivery:complete', i)
    end
    
    LS.UI:Notify('Összes delivery kész!', 'success')
end)
```
