# ls_marker

Az **ls_marker** egy DUI-alapú marker rendszer FiveM fejlesztéshez, amely lehetővé teszi 3D markerek, parkolóhelyek, waypoint-ok létrehozását és kezelését programozottan.

## Áttekintés

A resource egy teljes marker kezelő rendszert biztosít amely DUI-val rajzolódik ki, testreszabható markerek megjelenítéséhez a játékban.


### Marker típusok

  waypoint \
  mark \
  ground \
  groundpoly 

## API Exportok

### Marker létrehozás

#### CreateMarker

Általános marker létrehozó függvény.
Ez visszaadja a marker instance-t amivel lehet törölni és egyebeket!
```lua
---@param data table Marker konfiguráció
---@return table|nil Marker instance vagy nil ha sikertelen
exports.ls_marker:CreateMarker(data)
```

**Példa:**
```lua
local parkingSpot = exports.ls_marker:CreateMarker({
  id = "pd_parking_1",
  type = "ground",
  position = vector3(441.58, -981.23, 25.70),
  width = 17.2,
  height = 13.2,
  heading = 90.0,
  lineColor = "#e5e7eb",
  activeBorderColor = "#69db7c",
  activeBackgroundColor = "#40c057",
  errorBorderColor = "#ffa94d",
  errorBackgroundColor = "#fd7e14",
  onVehicleParked = function(markerId, vehicle)
    print("Jármű beparkolva:", vehicle)
  end,
  onVehicleExited = function(markerId, vehicle)
    print("Jármű elhagyta a helyet:", vehicle)
  end
})
```

### Marker kezelés

#### GetMarker

Marker lekérdezése ID alapján.

```lua
---@param id string Marker ID
---@return table|nil Marker instance
exports.ls_marker:GetMarker(id)
```

#### GetAllMarkers

Összes marker lekérdezése.

```lua
---@return table Összes marker
exports.ls_marker:GetAllMarkers()
```

#### EnableMarker / DisableMarker

Marker engedélyezése vagy letiltása.

```lua
exports.ls_marker:EnableMarker(id)   -- Engedélyezés
exports.ls_marker:DisableMarker(id)  -- Letiltás
```

#### DeleteMarker

Marker törlése.

```lua
---@param id string Marker ID
---@return boolean Sikeres törlés
exports.ls_marker:DeleteMarker(id)
```

**Példa:**
```lua
-- Temporary marker törlése 30 másodperc után
local tempMarker = exports.ls_marker:CreateWaypoint({
  id = "temp_marker",
  position = GetEntityCoords(PlayerPedId())
})

SetTimeout(30000, function()
  exports.ls_marker:DeleteMarker("temp_marker")
end)
```

### Marker frissítés


#### IsMarkerActive

Ellenőrzi, hogy a marker aktív-e (van-e teljesen beparkolva jármű).

```lua
---@param id string Marker ID
---@return boolean Aktív-e
exports.ls_marker:IsMarkerActive(id)
```

**Példa:**
```lua
-- Parkoló rendszer
RegisterNetEvent("parking:checkSpot")
AddEventHandler("parking:checkSpot", function(spotId)
  if exports.ls_marker:IsMarkerActive(spotId) then
    LS.UI:Notify("Ez a hely foglalt!", "error")
  else
    LS.UI:Notify("Hely szabad!", "success")
  end
end)
```

## Gyakorlati példák

### Rendőrségi parkoló rendszer

```lua
local pdParkingSpots = {
  { id = "pd_spot_1", pos = vector3(441.58, -981.23, 25.70), heading = 90.0 },
  { id = "pd_spot_2", pos = vector3(441.58, -986.50, 25.70), heading = 90.0 },
  { id = "pd_spot_3", pos = vector3(441.58, -991.77, 25.70), heading = 90.0 }
}

CreateThread(function()
  for _, spot in ipairs(pdParkingSpots) do
    exports.ls_marker:CreateGround({
      id = spot.id,
      position = spot.pos,
      width = 17.2,
      height = 13.2,
      heading = spot.heading,
      streamDistance = 100.0,
      lineColor = "#3b82f6",
      activeBorderColor = "#22c55e",
      activeBackgroundColor = "#16a34a",
      errorBorderColor = "#ef4444",
      errorBackgroundColor = "#dc2626",
      onVehicleParked = function(markerId, vehicle)
        local plate = GetVehicleNumberPlateText(vehicle)
        TriggerServerEvent("pd:vehicleParked", markerId, plate)
      end,
      onVehicleExited = function(markerId, vehicle)
        TriggerServerEvent("pd:vehicleLeft", markerId)
      end
    })
  end
end)
```

### Verseny checkpoint rendszer

```lua
local raceCheckpoints = {
  vector3(100.0, 200.0, 30.0),
  vector3(200.0, 300.0, 35.0),
  vector3(300.0, 400.0, 32.0),
  vector3(400.0, 500.0, 28.0)
}

local currentCheckpoint = 1

function StartRace()
  for i, pos in ipairs(raceCheckpoints) do
    exports.ls_marker:CreateWaypoint({
      id = "checkpoint_" .. i,
      position = pos,
      label = i .. ". CHECKPOINT",
      color = i == 1 and "#ffd43b" or "#64748b"
    })
    
    if i > 1 then
      exports.ls_marker:DisableMarker("checkpoint_" .. i)
    end
  end
end

function ReachCheckpoint(checkpointNum)
  exports.ls_marker:DeleteMarker("checkpoint_" .. checkpointNum)
  
  if checkpointNum < #raceCheckpoints then
    exports.ls_marker:EnableMarker("checkpoint_" .. (checkpointNum + 1))
    exports.ls_marker:UpdateMarkerVisual("checkpoint_" .. (checkpointNum + 1), "color", "#ffd43b")
  else
    LS.UI:Notify("Verseny befejezve!", "success")
  end
end
```

### Dinamikus delivery parkolók

```lua
local deliveryZones = {}

RegisterNetEvent("delivery:createZone")
AddEventHandler("delivery:createZone", function(zoneId, corners)
  deliveryZones[zoneId] = exports.ls_marker:CreateGroundPoly({
    id = zoneId,
    corners = corners,
    label = "DELIVERY ZONE",
    iconName = "IconTruck",
    lineColor = "#eab308",
    activeBorderColor = "#22c55e",
    activeBackgroundColor = "#16a34a",
    streamDistance = 200.0,
    onVehicleParked = function(markerId, vehicle)
      -- Ellenőrizd, hogy megfelelő típusú jármű-e
      local model = GetEntityModel(vehicle)
      if IsThisModelATruck(model) then
        TriggerServerEvent("delivery:complete", markerId)
      else
        LS.UI:Notify("Csak teherautókkal lehet szállítani!", "error")
      end
    end
  })
end)

RegisterNetEvent("delivery:removeZone")
AddEventHandler("delivery:removeZone", function(zoneId)
  exports.ls_marker:DeleteMarker(zoneId)
  deliveryZones[zoneId] = nil
end)
```

## Technikai részletek

### DUI Rendering

A resource React + TypeScript alapú NUI-t használ a markerek megjelenítéséhez. Minden marker típus egy külön React komponenst használ:

- **waypoint.tsx** - Távolság-alapú körök
- **MarkRoute.tsx** - Nyíl és gyémánt formák Tabler ikonokkal
- **GroundRoute.tsx** - Téglalap alakú parkolók
- **GroundPolyRoute.tsx** - Polygon alapú parkolók

### Kerék detektálás

A ground és groundpoly markerek minden kerék pozícióját ellenőrzik:
- Minimum 4 kereket igényel
- Minden keréknek a marker területén belül kell lennie
- Z koordináta tolerancia figyelembe vétellel

### Színátmenetek

A parkoló markerek automatikusan váltják a színüket:
- **Fehér/Szürke** - Üres
- **Zöld** - Minden kerék bent (teljesen beparkolva)
- **Narancssárga** - Részben bent (nem minden kerék)
