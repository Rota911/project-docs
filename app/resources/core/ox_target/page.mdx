# ox_target

Az **ox_target** egy target-based interakciós rendszer.

## Áttekintés

Eye-target rendszer entity-kkel és zónákkal való interakcióhoz.

### Főbb funkciók

- 🎯 **Entity targeting** - Ped, vehicle, object
- 📍 **Zone targeting** - Box, sphere, poly zones
- 👁️ **Láthatóság** - Distance és line of sight check
- 🔒 **Jogosultságok** - Job és item alapú
- 🎨 **Ikon rendszer** - FontAwesome ikonok

## Telepítés

### Függőségek

```lua
dependencies {
    'ls_core'
}
```

### server.cfg

```cfg
ensure ls_core
ensure ox_target
```

## Használat

### Entity target

```lua
-- Ped target
exports.ox_target:addLocalEntity(ped, {
    {
        name = 'talk_to_ped',
        icon = 'fas fa-comments',
        label = 'Beszélgetés',
        onSelect = function(data)
            print('Beszélgetés kezdése')
        end
    }
})

-- Vehicle target
exports.ox_target:addGlobalVehicle({
    {
        name = 'enter_vehicle',
        icon = 'fas fa-car',
        label = 'Beszállás',
        canInteract = function(entity, distance, coords, name)
            return not IsPedInAnyVehicle(PlayerPedId(), false)
        end,
        onSelect = function(data)
            TaskEnterVehicle(PlayerPedId(), data.entity, -1, -1, 1.0, 1, 0)
        end
    }
})
```

### Zone target

```lua
-- Box zone
exports.ox_target:addBoxZone({
    name = 'shop_zone',
    coords = vector3(25.0, -1347.0, 29.5),
    size = vector3(2, 2, 2),
    rotation = 45,
    options = {
        {
            name = 'open_shop',
            icon = 'fas fa-shopping-cart',
            label = 'Bolt megnyitása',
            onSelect = function()
                TriggerEvent('shop:open')
            end
        }
    }
})

-- Sphere zone
exports.ox_target:addSphereZone({
    name = 'atm_zone',
    coords = vector3(147.0, -1035.0, 29.3),
    radius = 1.5,
    options = {
        {
            name = 'use_atm',
            icon = 'fas fa-credit-card',
            label = 'ATM használata',
            onSelect = function()
                TriggerEvent('bank:openATM')
            end
        }
    }
})
```

## API Export-ok

### Client

```lua
-- Entity target hozzáadás
exports.ox_target:addLocalEntity(entity, options)
exports.ox_target:addEntity(netId, options)

-- Global entity target
exports.ox_target:addGlobalPed(options)
exports.ox_target:addGlobalVehicle(options)
exports.ox_target:addGlobalObject(options)
exports.ox_target:addGlobalPlayer(options)

-- Model target
exports.ox_target:addModel(models, options)

-- Zone target
exports.ox_target:addBoxZone(data)
exports.ox_target:addSphereZone(data)
exports.ox_target:addPolyZone(data)

-- Target eltávolítás
exports.ox_target:removeLocalEntity(entity, names)
exports.ox_target:removeZone(name)
exports.ox_target:removeModel(models, names)
```

## Példák

### Bolt rendszer

```lua
local shops = {
    {coords = vector3(25.0, -1347.0, 29.5), label = '247 Supermarket'},
    {coords = vector3(373.0, 325.0, 103.6), label = '247 Supermarket'},
}

for i, shop in ipairs(shops) do
    exports.ox_target:addBoxZone({
        name = 'shop_' .. i,
        coords = shop.coords,
        size = vector3(2, 2, 2),
        options = {
            {
                name = 'open_shop',
                icon = 'fas fa-shopping-cart',
                label = shop.label,
                onSelect = function()
                    TriggerEvent('shop:open', i)
                end
            }
        }
    })
end
```
