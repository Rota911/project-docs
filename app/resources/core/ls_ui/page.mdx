# ls_ui

## Client oldal

### TextUI

TextUI elem megjelenítése.

```lua
---@param data TextUI | TextUI[]
LS.UI:TextUI({
    id = 'example_textui',
    text = 'Nyomj [E]-t az interakcióhoz',
    position = 'center-bottom',
    icon = 'hand-paper',
    duration = 5000
})
```

### HideTextUI

TextUI elem elrejtése.

```lua
---@param id string | string[]
LS.UI:HideTextUI('example_textui')
```

### PatchTextUI

TextUI elem módosítása.

```lua
---@param data TextUI
LS.UI:PatchTextUI({
    id = 'example_textui',
    text = 'Módosított szöveg'
})
```

### HideAllTextUIs

Összes TextUI elem elrejtése.

```lua
LS.UI:HideAllTextUIs()
```

### Notification

Értesítés megjelenítése.

```lua
---@param data Notification
LS.UI:Notification({
    title = 'Siker',
    message = 'Az akcio sikeresen vegzodott',
    type = 'success',
    duration = 3000
})
```

### InputDialog

Beviteli dialógus megjelenítése.

```lua
---@param data InputDialog
---@return string|nil
local input = LS.UI:InputDialog({
    title = 'Szoveg bevitele',
    placeholder = 'Irj valamit...',
    maxLength = 50
})
```

### CloseInputDialog

Beviteli dialógus bezárása.

```lua
LS.UI:CloseInputDialog()
```

### ScreenFadeOut

Képernyő elhalványítása.

```lua
---@param duration number Elhalványítás időtartama (ms)
---@param headerMessage string|nil Fejléc üzenet
LS.UI:ScreenFadeOut(1000, 'Betöltés...')
```

### ScreenFadeIn

Képernyő megjelenítése.

```lua
---@param duration number Megjelenítés időtartama (ms)
LS.UI:ScreenFadeIn(1000)
```

## Szerver oldal

### Notification

Értesítés megjelenítése egy játékosnak.

```lua
---@param playerSrc number Játékos azonosítója
---@param data Notification
LS.UI:Notification(1, {
    title = 'Siker',
    message = 'Az akcio sikeresen vegzodott',
    type = 'success',
    duration = 3000
})
```

### InteractionNotification

Interakciós értesítés megjelenítése.

```lua
---@param playerSrc number Játékos azonosítója
---@param data InteractionNotification
---@return boolean
LS.UI:InteractionNotification(1, {
    title = 'Nyomj E-t',
    description = 'Az interakciohoz',
    icon = 'hand-paper'
})
```

### CloseInteractionNotification

Interakciós értesítés bezárása.

```lua
---@param playerSrc number Játékos azonosítója
LS.UI:CloseInteractionNotification(1)
```

## Adatstruktúrák

### TextUI

```lua
{
    id = 'example_id',              -- UI elem egyedi azonosítója
    text = 'UI szöveg',             -- Megjelenítendő szöveg
    position = 'center-bottom',     -- Pozíció
    icon = 'hand-paper',            -- Ikon (opcionális)
    duration = 5000                 -- Megjelenítés időtartama ms-ban (opcionális)
}
```

### Notification

```lua
{
    title = 'Cím',                  -- Értesítés címe
    message = 'Üzenet szövege',     -- Értesítés üzenete
    type = 'success',               -- Típus: 'success', 'error', 'info', 'warning'
    duration = 3000                 -- Megjelenítés időtartama ms-ban
}
```

### InputDialog

```lua
{
    title = 'Bevitel cím',          -- Dialógus címe
    placeholder = 'Placeholder',    -- Placeholder szöveg
    maxLength = 50,                 -- Maximum karakterhossz
    default = 'Alapértelmezett'     -- Alapértelmezett érték (opcionális)
}
```

### InteractionNotification

```lua
{
    title = 'Nyomj E-t',            -- Értesítés címe
    description = 'Interakcióhoz',  -- Leírás
    icon = 'hand-paper'             -- Ikon
}
```
