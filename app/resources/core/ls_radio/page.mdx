# ls_radio

Rádió kommunikációs rendszer React alapú UI-jal, frekvencia kezeléssel és kormányzati csatornákkal.

## Áttekintés

Az `ls_radio` egy rádió rendszert biztosít a játékosok számára, amely lehetővé teszi frekvencia alapú kommunikációt. Tartalmaz egy NUI-alapú rádió felületet, szerver oldali csatorna kezelést és előre definiált kormányzati csatornákat (rendőrség, mentők, stb.).

## Függőségek

- `ls_core`
- `oxmysql`

## Fájl struktúra

```
ls_radio/
  client/
    main.lua               -- Kliens oldali rádió kezelés
    modules/
      *.lua                -- Kliens modulok
  configs/
    govChannels.lua        -- Kormányzati csatorna definíciók
  server/
    main.lua               -- Szerver oldali csatorna és frekvencia kezelés
  web/
    index.html             -- Rádió NUI felület
    index.js               -- Rádió UI logika
```

## Főbb funkciók

- 📻 **Frekvencia alapú kommunikáció** — Játékosok frekvenciákra csatlakozhatnak és kommunikálhatnak
- 🏛️ **Kormányzati csatornák** — Előre definiált csatornák állami szervezeteknek
- 🎨 **NUI felület** — Vizuális rádió interfész frekvencia váltással
- 🔗 **VOIP integráció** — Az `ls_voip` resource használja a csatorna konfigurációkat

## Kormányzati csatornák

A `configs/govChannels.lua` fájl definiálja az előre beállított rádió csatornákat, amelyeket az állami szervezetek használnak. Ez a konfiguráció az `ls_voip` resource számára is elérhető:

```lua
-- Az ls_voip közvetlenül betölti:
-- @ls_radio/configs/govChannels.lua
```

A kormányzati csatornák tipikusan a következő szervezetekhez tartoznak:
- Rendőrség
- Mentőszolgálat
- Tűzoltóság
- Egyéb állami szervek

## NUI

A rádió felület egy egyszerű HTML/JS alapú NUI, amely lehetővé teszi:
- Frekvencia beállítását
- Csatorna váltást
- Hangerő szabályozást
- Rádió be-/kikapcsolást

## Integráció

### ls_voip

Az `ls_voip` az `ls_radio` kormányzati csatorna konfigurációját használja a rádió hangkommunikációhoz:

```lua
-- ls_voip fxmanifest.lua
shared_scripts {
    '@ls_radio/configs/govChannels.lua'
}
```

### ls_core

A resource az `ls_core` init script-et használja a framework funkciókhoz:

```lua
shared_scripts {
    '@ls_core/init.lua',
}
```

## Jó tudni

- A rádió frekvenciák szerver oldalon szinkronizálódnak az `oxmysql` adatbázison keresztül
- A `use_experimental_fxv2_oal` flag engedélyezve van a fejlett audio kezeléshez
- A kormányzati csatornák konfigurációja megosztott az `ls_voip` és `ls_radio` között
- A rádió UI PNG képeket használ a vizuális elemekhez
