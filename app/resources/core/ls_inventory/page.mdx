# ls_inventory

Az **ls_inventory** egy grid-alapú inventory rendszer item kezeléssel, tárolókkal, equipment rendszerrel és hook rendszerrel.

## Áttekintés

Modern, grid-alapú inventory rendszer React frontend-tel, szerver oldali instance-alapú API-val és hook rendszerrel.

### Főbb funkciók

- 🎒 **Grid inventory** — Grid-alapú (x,y) slot rendszer több inventory típussal
- 📦 **Tárolók** — Stash, container, backpack rendszer
- 🔫 **Equipment** — Fegyver és felszerelés slot rendszer
- 🔄 **Item kezelés** — Használat, drop, give, stack, split
- 🪝 **Hook rendszer** — Külső resource-ok számára pre-action interceptorok
- 🎯 **Instance API** — OOP-alapú PlayerInventory, Inventory és Item instance-ok
- ⏱️ **Decay & Uses** — Item lejárat és használati számláló
- 📊 **Metadata** — Egyedi item tulajdonságok

## Függőségek

- `ls_core`
- `oxmysql`

## Konfiguráció

### Alapértelmezett inventory-k

Minden karakter automatikusan kap egy `character` és egy `pocket` inventory-t:

```lua
Config.DefaultPlayerInventories = {
    {
        type = "character",
        label = "Inventory",
        gridWidth = 10,
        gridHeight = 10,
        middleInventory = true
    },
    {
        type = "pocket",
        label = "Pocket",
        gridWidth = 5,
        gridHeight = 5,
        middleInventory = true
    }
}
```
