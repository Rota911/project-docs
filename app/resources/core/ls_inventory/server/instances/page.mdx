# Instances

A szerver oldali instance rendszer OOP-alapú API-t biztosít az inventory kezeléséhez. Három fő instance típus létezik:

- **PlayerInventoryInstance** — Egy játékos összes inventory-ját kezeli (character, pocket, backpack)
- **InventoryInstance** — Egyetlen inventory kezelése
- **ItemInstance** — Egyetlen item kezelése

## PlayerInventoryInstance

A `GetPlayerInventory` export-tal kérhető le. Egy karakter összes inventory-ját összefogja.

```lua
local inventory = exports.ls_inventory:GetPlayerInventory(source)
```

**Paraméterek:**
- `source` (number) — Játékos server ID

**Visszatérési érték:** `PlayerInventoryInstance|nil`

---

### GetCharacterIdentifier

Karakter azonosító lekérése.

```lua
local charId = inventory:GetCharacterIdentifier()
```

**Visszatérési érték:** `string`

---

### GetInventoryIds

Összes inventory ID és típus lekérése.

```lua
local ids = inventory:GetInventoryIds()
```

**Visszatérési érték:** `table[]` — `{ id: string, type: string, label: string }[]`

```lua
local ids = inventory:GetInventoryIds()
for _, inv in ipairs(ids) do
    print(inv.id, inv.type, inv.label)
    -- "character:ABC123"  "character"  "Inventory"
    -- "pocket:ABC123"     "pocket"     "Pocket"
end
```

---

### GetInventory

Egy adott típusú inventory instance lekérése.

```lua
local inv = inventory:GetInventory(inventoryType)
```

**Paraméterek:**
- `inventoryType` (string|nil) — Inventory típus, alapértelmezett: `"character"`

**Visszatérési érték:** `InventoryInstance|nil`

```lua
local characterInv = inventory:GetInventory("character")
local pocketInv = inventory:GetInventory("pocket")
```

---

### GetAllInventories

Összes inventory instance lekérése típus szerint.

```lua
local all = inventory:GetAllInventories()
```

**Visszatérési érték:** `table<string, InventoryInstance>`

```lua
local all = inventory:GetAllInventories()
for invType, inv in pairs(all) do
    print(invType, inv:GetId(), inv:GetItemCount())
end
```

---

### GetItems

Összes item lekérése az összes inventory-ból. Opcionális kategória szűrővel.

```lua
local items = inventory:GetItems(category)
```

**Paraméterek:**
- `category` (string|nil) — Kategória szűrő (pl. `"weapon"`, `"consumable"`)

**Visszatérési érték:** `table<number, ItemInstance>`

```lua
local allItems = inventory:GetItems()
for itemId, item in pairs(allItems) do
    print(item:GetName(), item:GetCount())
end

local weapons = inventory:GetItems("weapon")
```

---

### GetItem

Egy adott nevű item lekérése (első találat).

```lua
local item = inventory:GetItem(itemName)
```

**Paraméterek:**
- `itemName` (string) — Item neve

**Visszatérési érték:** `ItemInstance|nil`

```lua
local water = inventory:GetItem('water')
if water then
    print(water:GetLabel(), water:GetCount())
end
```

---

### GetItemById

Item lekérése ID alapján.

```lua
local item = inventory:GetItemById(itemId)
```

**Paraméterek:**
- `itemId` (number) — Item ID

**Visszatérési érték:** `ItemInstance|nil`

```lua
local item = inventory:GetItemById(42)
if item then
    print(item:GetName())
end
```

---

### HasItem

Van-e adott item a játékosnál.

```lua
local hasItem = inventory:HasItem(itemName, amount)
```

**Paraméterek:**
- `itemName` (string) — Item neve
- `amount` (number|nil) — Elvárt mennyiség, alapértelmezett: `1`

**Visszatérési érték:** `boolean`

```lua
if inventory:HasItem('lockpick', 2) then
    print('Van legalább 2 lockpick')
end
```

---

### CountItem

Adott item teljes mennyisége az összes inventory-ban.

```lua
local count = inventory:CountItem(itemName)
```

**Paraméterek:**
- `itemName` (string) — Item neve

**Visszatérési érték:** `number`

```lua
local waterCount = inventory:CountItem('water')
print(("Összesen %d víz"):format(waterCount))
```

---

### CanCarryAmount

Mennyi fér még az adott itemből.

```lua
local canCarry, totalCapacity = inventory:CanCarryAmount(itemName)
```

**Paraméterek:**
- `itemName` (string) — Item neve

**Visszatérési érték:**
- `boolean` — Van-e hely
- `number` — Összes szabad kapacitás

```lua
local canCarry, capacity = inventory:CanCarryAmount('water')
if canCarry then
    print(("Kapacitás: %d"):format(capacity))
end
```

---

### AddItem

Item hozzáadása a játékos inventory-jához.

```lua
local result = inventory:AddItem(itemName, metadata, uses, amount)
```

**Paraméterek:**
- `itemName` (string) — Item neve
- `metadata` (table|nil) — Item metadata
- `uses` (number|nil) — Használati számláló
- `amount` (number|nil) — Mennyiség, alapértelmezett: `1`

**Visszatérési érték:** `table` — `{ ok: boolean, data: table|nil, msg: string|nil }`

```lua
local result = inventory:AddItem('water', nil, nil, 5)
if result.ok then
    print(("Hozzáadva: %d db"):format(result.data.totalAdded))
end

local result = inventory:AddItem('phone', { number = '555-1234' }, nil, 1)
```

---

### RemoveItem

Item eltávolítása ID alapján.

```lua
local result = inventory:RemoveItem(itemId, amount)
```

**Paraméterek:**
- `itemId` (number) — Item ID
- `amount` (number|nil) — Mennyiség (stackelhető itemeknél)

**Visszatérési érték:** `table` — `{ ok: boolean }`

```lua
local item = inventory:GetItem('water')
if item then
    local result = inventory:RemoveItem(item:GetId(), 1)
end
```

---

### Filter

Itemek szűrése egyedi callback-kel.

```lua
local items = inventory:Filter(callback)
```

**Paraméterek:**
- `callback` (function) — `fun(item: ItemInstance): boolean`

**Visszatérési érték:** `table<number, ItemInstance>`

```lua
local stackableItems = inventory:Filter(function(item)
    return item:IsStackable()
end)

local heavyItems = inventory:Filter(function(item)
    return item:GetWeight() > 1000
end)
```

---

### Find

Első item megtalálása egyedi callback-kel.

```lua
local item = inventory:Find(callback)
```

**Paraméterek:**
- `callback` (function) — `fun(item: ItemInstance): boolean`

**Visszatérési érték:** `ItemInstance|nil`

```lua
local firstUsable = inventory:Find(function(item)
    return item:IsUsable()
end)

if firstUsable then
    print(("Használható: %s"):format(firstUsable:GetLabel()))
end
```

---

### ForEach

Iteráció az összes itemen.

```lua
inventory:ForEach(callback)
```

**Paraméterek:**
- `callback` (function) — `fun(item: ItemInstance)`

```lua
inventory:ForEach(function(item)
    print(("%s x%d"):format(item:GetName(), item:GetCount()))
end)
```

---

### Serialize

Instance szerializálása.

```lua
local data = inventory:Serialize()
```

**Visszatérési érték:** `table` — `{ characterIdentifier, inventories, itemCount, totalWeight }`

## InventoryInstance

Egyetlen inventory kezelése. A `GetInventory` export-tal vagy a `PlayerInventoryInstance:GetInventory()` metódussal kérhető le.

```lua
local inv = exports.ls_inventory:GetInventory(inventoryId)
```

**Paraméterek:**
- `inventoryId` (string) — Inventory azonosító (pl. `"character:ABC123"`, `"stash:police"`)

**Visszatérési érték:** `InventoryInstance|nil`

Az `InventoryInstance` ugyanazokkal a metódusokkal rendelkezik mint a `PlayerInventoryInstance`, de egyetlen inventory-ra vonatkoznak:

| Metódus | Visszatérési érték | Leírás |
|---|---|---|
| `GetId()` | `string` | Inventory ID |
| `GetCharacterIdentifier()` | `string` | Karakter azonosító |
| `GetItems(category?)` | `table<number, ItemInstance>` | Itemek lekérése |
| `GetItem(itemName)` | `ItemInstance\|nil` | Első item név alapján |
| `GetItemById(itemId)` | `ItemInstance\|nil` | Item ID alapján |
| `HasItem(itemName, amount?)` | `boolean` | Van-e item |
| `CountItem(itemName)` | `number` | Item mennyiség |
| `CanCarryAmount(itemName)` | `boolean, number\|nil` | Szabad kapacitás |
| `AddItem(itemName, metadata?, uses?, amount?)` | `table` | Item hozzáadás |
| `RemoveItem(itemId, amount?)` | `table` | Item eltávolítás (ID) |
| `RemoveItemByName(itemName, amount)` | `table` | Item eltávolítás (név) |
| `Clear()` | `table` | Inventory kiürítés |
| `GetItemCount()` | `number` | Item darabszám |
| `GetTotalWeight()` | `number` | Teljes súly |
| `IsEmpty()` | `boolean` | Üres-e |
| `GetItemsByFlag(flag)` | `table<number, ItemInstance>` | Itemek flag alapján |
| `GetItemsByName(itemName)` | `table<number, ItemInstance>` | Itemek név alapján |
| `GetExpiredItems()` | `table<number, ItemInstance>` | Lejárt itemek |
| `GetDepletedItems()` | `table<number, ItemInstance>` | Elhasznált itemek |
| `RemoveExpiredItems()` | `number` | Lejárt itemek eltávolítása |
| `RemoveDepletedItems()` | `number` | Elhasznált itemek eltávolítása |
| `Filter(callback)` | `table<number, ItemInstance>` | Szűrés |
| `Find(callback)` | `ItemInstance\|nil` | Keresés |
| `ForEach(callback)` | — | Iteráció |
| `Serialize()` | `table` | Szerializálás |

```lua
local stashInv = exports.ls_inventory:GetInventory('stash:police_armory')
if stashInv then
    stashInv:ForEach(function(item)
        print(item:GetName(), item:GetCount())
    end)
end
```

### Use

Item használata (csökkenti a uses számlálót, ha elfogyott eltávolítja).

```lua

```

**Visszatérési érték:** `table` — `{ ok: boolean, data: { consumed: boolean } }`

```lua
local item = inventory:GetItem('medkit')
if item and item:IsUsable() and not item:IsDepleted() then
    local result = item:Use()
    if result.ok and result.data.consumed then
        print('Medkit elhasználva és eltávolítva')
    end
end
```
