# Server

Az ls_inventory szerver oldala tartalmazza az inventory kezelés teljes üzleti logikáját, az instance-alapú API-t, az event rendszert és a hook rendszert.

## Felépítés

| Réteg | Leírás |
|---|---|
| [Exports](/resources/core/ls_inventory/server/exports) | Külső resource-ok számára elérhető függvények |
| [Instances](/resources/core/ls_inventory/server/instances) | OOP-alapú PlayerInventory, Inventory és Item instance-ok |
| [Events](/resources/core/ls_inventory/server/events) | Post-action event rendszer (itemAdded, itemRemoved, stb.) |
| [Hooks](/resources/core/ls_inventory/server/hooks) | Pre-action hook rendszer — akciók blokkolása vagy módosítása |

## Gyors példa

### Export-okkal

```lua
local player = LS:GetPlayer(source)
local charId = player:GetActiveCharacter().charId

local result = exports.ls_inventory:AddItem(charId, 'water', nil, nil, 5)
if result.ok then
    print(("Hozzáadva: %d/%d"):format(result.data.totalAdded, result.data.totalRequested))
end

local hasWater = exports.ls_inventory:HasItem(charId, 'water', 1)
```

### Instance-okkal

```lua
local inventory = exports.ls_inventory:GetPlayerInventory(source)

inventory:AddItem('water', nil, nil, 5)

if inventory:HasItem('water', 3) then
    inventory:RemoveItemByName('water', 3)
end

local item = inventory:GetItem('water')
if item then
    print(item:GetLabel(), item:GetCount())
end
```

### Hook-okkal

```lua
exports.ls_inventory:registerHook('addItem', function(payload)
    if payload.itemName == 'contraband' then
        return false
    end
end)
```
