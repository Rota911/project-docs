# Client

## Exports

### GetItemDef

Item definíció lekérése a cache-ből.

```lua
local itemDef = exports.ls_inventory:GetItemDef(itemName)
```

**Paraméterek:**
- `itemName` (string) — Az item neve

**Visszatérési érték:** `table|nil` — Az item definíció vagy nil

```lua
local def = exports.ls_inventory:GetItemDef('water')
if def then
    print(def.label)    -- "Víz"
    print(def.weight)   -- 500
    print(def.category) -- "consumable"
end
```

---

### GetAllItems

Összes item definíció lekérése.

```lua
local items = exports.ls_inventory:GetAllItems()
```

**Visszatérési érték:** `table` — Összes item definíció

```lua
local allItems = exports.ls_inventory:GetAllItems()
for name, def in pairs(allItems) do
    print(name, def.label)
end
```

### GetDropsNearby

A játékos körzetében lévő összes aktív drop lekérése (`Config.DropSystem.LoadDistance` sugarán belül).

```lua
local drops = exports.ls_inventory:GetDropsNearby()
```

**Visszatérési érték:** `DropClientData[]|false` — Dropok listája, vagy `false` ha nincs a közelben egy sem, illetve ha nincs bejelentkezett karakter.

```lua
local drops = exports.ls_inventory:GetDropsNearby()
if drops then
    for _, drop in ipairs(drops) do
        print(drop.dropId, drop.label, drop.inventoryId)
    end
end
```

---

### GetDropAtCoord

Adott koordináta körzetében lévő dropok lekérése.

```lua
local drops = exports.ls_inventory:GetDropAtCoord(coords, radius)
```

**Paraméterek:**
- `coords` (vector3) — Középpont koordináta
- `radius` (number) — Keresési sugár méterben

**Visszatérési érték:** `DropClientData[]|false` — Dropok listája, vagy `false` ha nincs találat.

```lua
local coords = GetEntityCoords(cache.ped)
local drops = exports.ls_inventory:GetDropAtCoord(coords, 5.0)
if drops then
    for _, drop in ipairs(drops) do
        print(("Drop találat: %s (%.1fm)"):format(drop.label, #(coords - drop.coords)))
    end
end
```

> Részletes Drop System dokumentáció: [Drop System](/resources/core/ls_inventory/server/drops)

### Item Use Events

Az inventory a `useItem` callback után item típus alapján client event-et triggerel:

| Event | Leírás |
|---|---|
| `ls_inventory:useConsumable` | Fogyasztható item (étel, ital) |
| `ls_inventory:useMedical` | Gyógyszer, elsősegély |
| `ls_inventory:equipWeapon` | Fegyver felszerelése |
| `ls_inventory:equipArmor` | Páncél felszerelése |
| `ls_inventory:usePhone` | Telefon használata |
| `ls_inventory:useTool` | Eszköz használata |
| `ls_inventory:useRepairKit` | Javító készlet |
| `ls_inventory:useFlashlight` | Zseblámpa |
| `ls_inventory:useBinoculars` | Távcső |
| `ls_inventory:useRadio` | Rádió |

Minden event megkapja a `data` táblát `itemName` és `metadata` mezőkkel:

```lua
RegisterNetEvent('ls_inventory:useConsumable', function(data)
    print(data.itemName)  -- pl. 'water'
    print(data.metadata)  -- item metadata
end)
```

Ezek az events local event-eket is triggerelnek amire hallgathatsz:

```lua
AddEventHandler('ls_inventory:weaponEquipped', function(itemName, weaponHash)
    print(("Fegyver equipálva: %s (%s)"):format(itemName, weaponHash))
end)

AddEventHandler('ls_inventory:toolUsed', function(itemName, metadata)
    print(("Eszköz használva: %s"):format(itemName))
end)
```
