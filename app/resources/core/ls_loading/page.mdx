# ls_loading

Az **ls_loading** egy testreszabható loading screen resource web-based UI-val.

## Áttekintés

Modern loading képernyő csatlakozáskor background videóval, képekkel és zenével.

### Főbb funkciók

- 🎥 **Video háttér** - MP4 videók lejátszása
- 🖼️ **Kép háttér** - Slideshow vagy statikus kép
- 🎵 **Háttérzene** - Audio lejátszás
- 📊 **Progress bar** - Betöltés állapot
- 🎨 **Testreszabható** - CSS és HTML szerkesztés
- ℹ️ **Szerverinfó** - Név, játékosszám, szabályok

## Telepítés

### server.cfg

```cfg
loadscreen 'https://cfx-nui-ls_loading/web/index.html'
ensure ls_loading
```

## Konfiguráció

### web/config.js

```js
const Config = {
    // Szerver név
    serverName: 'LS Server',
    
    // Háttér típus: 'video', 'image', 'slideshow'
    backgroundType: 'video',
    
    // Video fájl (ha type = 'video')
    backgroundVideo: 'background.mp4',
    
    // Képek (ha type = 'image' vagy 'slideshow')
    backgroundImages: [
        'bg1.jpg',
        'bg2.jpg',
        'bg3.jpg'
    ],
    
    // Slideshow interval (ms)
    slideshowInterval: 5000,
    
    // Háttérzene
    music: {
        enabled: true,
        file: 'music.mp3',
        volume: 0.3  // 0.0 - 1.0
    },
    
    // Színek
    colors: {
        primary: '#00ff88',
        secondary: '#ffffff',
        background: 'rgba(0, 0, 0, 0.7)'
    },
    
    // Szabályok
    rules: [
        'Tiszteld a többi játékost',
        'Ne használj cheateket',
        'Tartsd be a RP szabályokat',
        'Segíts az új játékosoknak'
    ],
    
    // Discord link
    discord: 'https://discord.gg/yourserver',
    
    // Weboldal
    website: 'https://yourserver.com',
    
    // Progress bar szöveg
    loadingTexts: [
        'Kapcsolódás a szerverhez...',
        'Adatok betöltése...',
        'Majdnem kész...',
        'Üdvözlünk!'
    ]
};
```
