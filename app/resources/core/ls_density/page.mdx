# ls_density

Az **ls_density** egy NPC és jármű populáció kezelő resource.

## Áttekintés

Dinamikus density kezelés a szerver teljesítményének és élmény optimalizálásához.

### Főbb funkciók

- 🚗 **Jármű density** - Járműforgalom szabályozása
- 👥 **NPC density** - Gyalogos populáció kezelés
- 🚓 **Parked vehicles** - Parkolt járművek kezelése
- 🌃 **Scenario density** - Ambient scenarios kezelése
- ⚙️ **Zóna alapú** - Különböző területeken különböző beállítások
- 🎮 **Teljesítmény** - FPS optimalizálás

## Használat

### Density beállítása

```lua
-- Globális density
exports.ls_density:SetVehicleDensity(0.5)  -- 50% járművek
exports.ls_density:SetPedDensity(0.3)      -- 30% gyalogosok
exports.ls_density:SetParkedDensity(0.7)   -- 70% parkolt járművek

-- Scenarios toggle
exports.ls_density:SetScenarios(true)  -- Bekapcsolás
exports.ls_density:SetScenarios(false) -- Kikapcsolás

-- Teljes forgalom letiltás
exports.ls_density:SuppressTraffic(true)
```

### Zóna alapú kezelés

```lua
-- Zóna hozzáadása runtime-ban
exports.ls_density:AddZone({
    name = 'Custom Zone',
    coords = vector3(100.0, 200.0, 30.0),
    radius = 250.0,
    settings = {
        vehicles = 0.0,
        peds = 0.0,
        parked = 0.0,
        scenarios = false,
        suppressTraffic = true
    }
})

-- Zóna eltávolítása
exports.ls_density:RemoveZone('Custom Zone')

-- Zóna módosítása
exports.ls_density:UpdateZone('Custom Zone', {
    vehicles = 0.5,
    peds = 0.5
})
```

## Natív funkciók

Az ls_density a következő natívokat használja:

### Jármű density

```lua
-- Járművek spawnjának szabályozása
SetVehicleDensityMultiplierThisFrame(density)

-- Random járművek
SetRandomVehicleDensityMultiplierThisFrame(density)

-- Parkolt járművek
SetParkedVehicleDensityMultiplierThisFrame(density)
```

### NPC density

```lua
-- Gyalogos density
SetPedDensityMultiplierThisFrame(density)

-- Scenario ped density
SetScenarioPedDensityMultiplierThisFrame(density, scenario)
```

### Forgalom tiltás

```lua
-- Összes jármű spawnjának letiltása
SetVehicleDensityMultiplierThisFrame(0.0)
SetRandomVehicleDensityMultiplierThisFrame(0.0)
SetParkedVehicleDensityMultiplierThisFrame(0.0)

-- Összes NPC letiltása
SetPedDensityMultiplierThisFrame(0.0)
SetScenarioPedDensityMultiplierThisFrame(0.0, 0.0)
```

```lua
exports.ls_density:SetParkedDensity(density)
-- density: 0.0 - 1.0
```

#### Set Scenarios

```lua
exports.ls_density:SetScenarios(enabled)
-- enabled: true/false
```

#### Suppress Traffic

```lua
exports.ls_density:SuppressTraffic(suppress)
-- suppress: true/false
```

#### Add Zone

```lua
exports.ls_density:AddZone(zoneData)
```

#### Remove Zone

```lua
exports.ls_density:RemoveZone(zoneName)
```

#### Update Zone

```lua
exports.ls_density:UpdateZone(zoneName, settings)
```

#### Get Current Settings

```lua
local settings = exports.ls_density:GetCurrentSettings()
-- returns: {vehicles, peds, parked, scenarios, suppressTraffic}
```

Citizen.CreateThread(function()

### Dinamikus zóna kezelés

```lua
-- Játékosszám alapú density
Citizen.CreateThread(function()
    while true do
        Citizen.Wait(300000) -- 5 perc
        
        local playerCount = #GetActivePlayers()
        
        local density
        if playerCount > 64 then
            density = 0.2  -- Sok játékos = alacsony NPC
        elseif playerCount > 32 then
            density = 0.4
        else
            density = 0.6  -- Kevés játékos = több NPC
        end
        
        exports.ls_density:SetVehicleDensity(density)
        exports.ls_density:SetPedDensity(density * 0.8)
    end
end)
```

### Blacklist kezelés

```lua
-- Tiltott járművek eltávolítása
Citizen.CreateThread(function()
    while true do
        Citizen.Wait(5000)
        
        local vehicles = GetGamePool('CVehicle')
        for _, vehicle in ipairs(vehicles) do
            local model = GetEntityModel(vehicle)
            
            for _, blacklisted in ipairs(Config.BlacklistedVehicles) do
                if model == blacklisted then
                    DeleteEntity(vehicle)
                    break
                end
            end
        end
    end
end)
```

## Teljesítmény optimalizálás

### Frame-by-frame control

```lua
-- Optimalizált density control
Citizen.CreateThread(function()
    while true do
        local settings = exports.ls_density:GetCurrentSettings()
        
        -- Natívok hívása minden frame-ben
        SetVehicleDensityMultiplierThisFrame(settings.vehicles)
        SetRandomVehicleDensityMultiplierThisFrame(settings.vehicles)
        SetParkedVehicleDensityMultiplierThisFrame(settings.parked)
        SetPedDensityMultiplierThisFrame(settings.peds)
        SetScenarioPedDensityMultiplierThisFrame(settings.peds, settings.peds)
        
        if settings.suppressTraffic then
            SetVehicleDensityMultiplierThisFrame(0.0)
            SetRandomVehicleDensityMultiplierThisFrame(0.0)
            SetPedDensityMultiplierThisFrame(0.0)
        end
        
        Citizen.Wait(0)
    end
end)
```

## Troubleshooting

### Még mindig sok NPC

```lua
-- Ellenőrizd a beállításokat
local settings = exports.ls_density:GetCurrentSettings()
print(json.encode(settings))

-- Force 0 density
exports.ls_density:SuppressTraffic(true)
```

### FPS problémák

```lua
-- Csökkentsd a density-t
exports.ls_density:SetVehicleDensity(0.0)
exports.ls_density:SetPedDensity(0.0)
exports.ls_density:SetScenarios(false)
```
