# ls_density

### Density beállítása

```lua
-- Globális density
exports.ls_density:SetVehicleDensity(0.5)  -- 50% járművek
exports.ls_density:SetPedDensity(0.3)      -- 30% gyalogosok
exports.ls_density:SetParkedDensity(0.7)   -- 70% parkolt járművek

-- Scenarios toggle
exports.ls_density:SetScenarios(true)  -- Bekapcsolás
exports.ls_density:SetScenarios(false) -- Kikapcsolás

-- Teljes forgalom letiltás
exports.ls_density:SuppressTraffic(true)
```

### Zóna alapú kezelés

```lua
-- Zóna hozzáadása runtime-ban
exports.ls_density:AddZone({
    name = 'Custom Zone',
    coords = vector3(100.0, 200.0, 30.0),
    radius = 250.0,
    settings = {
        vehicles = 0.0,
        peds = 0.0,
        parked = 0.0,
        scenarios = false,
        suppressTraffic = true
    }
})

-- Zóna eltávolítása
exports.ls_density:RemoveZone('Custom Zone')

-- Zóna módosítása
exports.ls_density:UpdateZone('Custom Zone', {
    vehicles = 0.5,
    peds = 0.5
})
```


#### Set Vehicle Density

```lua
exports.ls_density:SetVehicleDensity(density)
-- density: 0.0 - 1.0
```

#### Set Ped Density

```lua
exports.ls_density:SetPedDensity(density)
-- density: 0.0 - 1.0
```

#### Set Parked Density

```lua
exports.ls_density:SetParkedDensity(density)

```

#### Set Scenarios

```lua
exports.ls_density:SetScenarios(enabled)
-- enabled: true/false
```

#### Suppress Traffic

```lua
exports.ls_density:SuppressTraffic(suppress)
-- suppress: true/false
```

#### Add Zone

```lua
exports.ls_density:AddZone(zoneData)
```

#### Remove Zone

```lua
exports.ls_density:RemoveZone(zoneName)
```

#### Update Zone

```lua
exports.ls_density:UpdateZone(zoneName, settings)
```

#### Get Current Settings

```lua
local settings = exports.ls_density:GetCurrentSettings()
-- returns: {vehicles, peds, parked, scenarios, suppressTraffic}
```
