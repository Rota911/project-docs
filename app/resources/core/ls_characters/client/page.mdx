# Client

## Events

| Event | Parameters | Description |
|---|---|---|
| `Core:Client:PlayerJoined` | `characters` | Fired when the player joins and characters are loaded |
| `Core:Client:OpenCharacterSelector` | `characters` | Fired to reopen the character selector |
| `Core:Client:CharacterLoaded` | `characterData` | Fired when a character is successfully loaded |
