# ls_characters

Az **ls_characters** egy modern karakter választó és kezelő rendszer web-based UI-val.

## Áttekintés

Teljes körű multi-karakter rendszer karakterek létrehozásával, törléssel és váltással.

### Főbb funkciók

- 👥 **Multi-karakter** - Több karakter játékosonként
- ✨ **Modern UI** - React-based responsive design
- 🎭 **Karakter preview** - 3D karakter megjelenítés
- 💾 **Perzisztens tárolás** - Database integráció
- 🔒 **Karakter limit** - Konfigurálható max karakterszám
- 🎨 **ls_appearance integráció** - Seamless megjelenés szerkesztés

## Telepítés

## Használat

### Karakter választó megnyitása

```lua
-- Export használat
exports.ls_characters:OpenSelector()

-- Automatikusan megnyílik bejelentkezéskor
```

### Karakter választás folyamata

1. **Bejelentkezés** - Játékos csatlakozik a szerverre
2. **Karakter lista** - Meglévő karakterek megjelenítése
3. **Válassz vagy hozz létre** - Karakter kiválasztása vagy új létrehozása
4. **Megjelenés** - ls_appearance megnyitása (új karakternél)
5. **Spawn** - Játékos spawnja a kiválasztott karakterrel

## Karakter struktúra

### Character data formátum

```lua
{
    charId = 1,                    -- Egyedi karakter ID
    playerId = 1,                  -- Player ID (külső kulcs)
    firstName = 'John',            -- Keresztnév
    lastName = 'Doe',              -- Vezetéknév
    dateOfBirth = '1990-01-01',    -- Születési dátum
    gender = 'male',               -- Nem
    nationality = 'Magyar',        -- Nemzetiség
    
    -- Játék adatok
    money = 5000,                  -- Készpénz
    bank = 10000,                  -- Bank egyenleg
    job = 'unemployed',            -- Munka
    jobGrade = 0,                  -- Munka rang
    gang = nil,                    -- Gang
    gangGrade = 0,                 -- Gang rang
    
    -- Pozíció
    coords = {
        x = -269.94,
        y = -955.14,
        z = 31.22,
        w = 205.0
    },
    
    -- Megjelenés
    appearance = {},               -- ls_appearance data
    
    -- Metadata
    isDead = false,                -- Halott állapot
    lastPlayed = '2024-01-07',     -- Utolsó játszás
    playTime = 120                 -- Játékidő (percek)
}
```

## API Export-ok

### Client

#### Open Selector

Karakter választó megnyitása:

```lua
exports.ls_characters:OpenSelector()
```

#### Close Selector

Karakter választó bezárása:

```lua
exports.ls_characters:CloseSelector()
```

#### Get Active Character

Aktív karakter lekérése:

```lua
local character = exports.ls_characters:GetActiveCharacter()
```

#### Get Characters

Összes karakter lekérése:

```lua
local characters = exports.ls_characters:GetCharacters()
```

### Server

#### Get Character

Karakter lekérése ID alapján:

```lua
local character = exports.ls_characters:GetCharacter(charId)
```

#### Get Player Characters

Játékos karaktereinek lekérése:

```lua
local characters = exports.ls_characters:GetPlayerCharacters(source)
```

#### Create Character

Új karakter létrehozása:

```lua
local charId = exports.ls_characters:CreateCharacter(source, data)
```

#### Delete Character

Karakter törlése:

```lua
local success = exports.ls_characters:DeleteCharacter(charId)
```

#### Update Character

Karakter frissítése:

```lua
exports.ls_characters:UpdateCharacter(charId, data)
```

### ls_appearance integráció

```lua
-- Új karakter létrehozásakor
RegisterNetEvent('ls_characters:createNew')
AddEventHandler('ls_characters:createNew', function(data)
    -- Karakter adatok validálás
    if not data.firstName or not data.lastName then
        return
    end
    
    -- ls_appearance megnyitása
    exports.ls_appearance:Open({
        type = 'character',
        allowModelChange = true,
        allowSave = false,  -- Még ne mentse, csak preview
        onSave = function(appearance)
            -- Karakter létrehozás appearance-szal
            TriggerServerEvent('ls_characters:finalizeCreation', data, appearance)
        end
    })
end)
```

## Gyakorlati példák

### Custom spawn logic

```lua
-- Server side
AddEventHandler('ls_characters:characterSelected', function(source, charId)
    local character = GetCharacter(source)
    
    -- Custom spawn koordináták munkánként
    local spawnCoords
    
    if character.job == 'police' then
        spawnCoords = vector4(425.1, -979.5, 30.7, 0.0)
    elseif character.job == 'ambulance' then
        spawnCoords = vector4(298.8, -584.6, 43.2, 0.0)
    else
        -- Mentett pozíció vagy default
        spawnCoords = character:GetCoords() or Config.SpawnCoords
    end
    
    TriggerClientEvent('ls_characters:spawn', source, spawnCoords)
end)
```

### Karakter limit check

```lua
-- Server side
RegisterNetEvent('ls_characters:checkLimit')
AddEventHandler('ls_characters:checkLimit', function()
    local source = source
    local characters = exports.ls_characters:GetPlayerCharacters(source)
    
    if #characters >= Config.MaxCharacters then
        TriggerClientEvent('ls_characters:limitReached', source)
        LS.UI:Notify(source, 'Elérted a maximum karakter limitet!', 'error')
    else
        TriggerClientEvent('ls_characters:openCreator', source)
    end
end)
```

### Karakter törlés confirmálással

### First character bonus

```lua
-- Server side
AddEventHandler('ls_characters:characterCreated', function(source, charId, data)
    local characters = exports.ls_characters:GetPlayerCharacters(source)
    
    -- Ha ez az első karakter
    if #characters == 1 then
        local character = GetCharacter(source)
        
        -- Starter csomag
        character:SetData('money', 10000)
        character:SetData('bank', 25000)
        
        -- Starter itemek (ha van inventory)
        exports.ls_inventory:AddItem(character.id, 'phone', 1)
        exports.ls_inventory:AddItem(character.id, 'id_card', 1)
        exports.ls_inventory:AddItem(character.id, 'driver_license', 1)
        
        LS.UI:Notify(source, 'Starter csomag átvéve!', 'success')
    end
end)
```

### Multi-karakter kezelés

```lua
-- Karakter váltás
RegisterNetEvent('ls_characters:switchCharacter')
AddEventHandler('ls_characters:switchCharacter', function()
    local source = source
    local currentChar = GetCharacter(source)
    
    if currentChar then
        -- Jelenlegi karakter mentése és unload
        currentChar:Save()
        currentChar:Unload()
    end
    
    -- Karakter választó megnyitása
    TriggerClientEvent('ls_characters:openSelector', source)
end)
```

### Karakter adatok validálás

```lua
-- Server side
local function ValidateCharacterData(data)
    -- Név ellenőrzés
    if not data.firstName or #data.firstName < 2 or #data.firstName > 50 then
        return false, 'Érvénytelen keresztnév'
    end
    
    if not data.lastName or #data.lastName < 2 or #data.lastName > 50 then
        return false, 'Érvénytelen vezetéknév'
    end
    
    -- Születési dátum
    local dob = os.time({
        year = data.year,
        month = data.month,
        day = data.day
    })
    
    local age = os.difftime(os.time(), dob) / (365.25 * 24 * 60 * 60)
    
    if age < 18 or age > 100 then
        return false, 'A karakternek 18-100 év között kell lennie'
    end
    
    -- Nem
    if data.gender ~= 'male' and data.gender ~= 'female' then
        return false, 'Érvénytelen nem'
    end
    
    return true
end

RegisterNetEvent('ls_characters:create')
AddEventHandler('ls_characters:create', function(data)
    local source = source
    
    local valid, error = ValidateCharacterData(data)
    if not valid then
        return LS.UI:Notify(source, error, 'error')
    end
    
    -- Karakter létrehozás
    local charId = exports.ls_characters:CreateCharacter(source, data)
end)
```

### NUI Callbacks

```lua
-- Karakter lista lekérése
RegisterNUICallback('getCharacters', function(data, cb)
    local characters = exports.ls_characters:GetCharacters()
    cb(characters)
end)

-- Karakter kiválasztása
RegisterNUICallback('selectCharacter', function(data, cb)

    cb('ok')

-- Új karakter létrehozása
RegisterNUICallback('createCharacter', function(data, cb)

RegisterNUICallback('deleteCharacter', function(data, cb)
    TriggerServerEvent('ls_characters:delete', data.charId)
    cb('ok')
end)
```

## Troubleshooting

### Karakterek nem töltődnek be

```lua
-- Ellenőrizd a database kapcsolatot
local characters = exports.ls_characters:GetPlayerCharacters(source)
print('Karakterek száma:', #characters)
```

### Spawn nem működik

```lua
-- Ellenőrizd a koordinátákat
print('Spawn coords:', Config.SpawnCoords)

-- Ellenőrizd a karakter coords-ot
local character = GetCharacter(source)
print('Character coords:', json.encode(character:GetCoords()))
```
