# ls_characters

Karakter választó és létrehozó rendszer React alapú UI-jal. Kezeli a karakter kiválasztást, létrehozást és a belépési folyamatot.

## Áttekintés

Az `ls_characters` felelős a játékos belépési folyamatáért: a loading screen után megnyitja a karakter választó felületet, ahol a játékos kiválaszthatja vagy létrehozhatja karakterét. A resource kamera rendszert, NUI-t és az `ls_core` Player osztályt használja a karakter kezeléshez.

## Függőségek

- `ls_core`
- `ls_appearance` (megjelenés adatok betöltése és mentése)

## Konfiguráció

### Slot-ok

A karakter slot-ok admin csoportonként konfigurálhatók:

```lua
Config.Slots = {
    ['admin'] = 3,  -- Adminok max karakter száma
}
```

### Specializációk

A karakter létrehozásnál választható karakter útvonalak:

```lua
Config.Specializations = {
    { id = 'citizen', label = 'Állampolgár' },
    { id = 'criminal', label = 'Bűnöző' },
}
```

## Belépési folyamat

1. Játékos csatlakozik → `Core:Client:PlayerJoined` event tüzel a karakterek listájával
2. Loading screen bezárása (ha aktív)
3. Kamera beállítása a karakter választó jelenethez
4. NUI megnyitása a karakter adatokkal
5. Játékos választ vagy létrehoz egy karaktert

```
Core:Client:PlayerJoined
    ↓
InitCharacterData(characters)
    ↓
Camera:Setup()
    ↓
NUI: setVisible + characterData
    ↓
Játékos választ / létrehoz
    ↓
Core:Client:CharacterLoaded
```

## Client oldal

### Karakter lista

A kliens formázza a karakterek adatait a NUI számára, beleértve a megjelenés adatok betöltését `ls_appearance`-ból:

```lua
{
    id = number,
    charId = number,
    first_name = string,
    last_name = string,
    middle_name = string,
    last_played = number,
    last_position = string,    -- Utca neve az utolsó pozíció alapján
    appearance = table         -- ls_appearance adat
}
```

### Karakter választó újranyitása

A karakter választó bármikor újranyitható szerver oldalról:

```lua
TriggerClientEvent('Core:Client:OpenCharacterSelector', playerSrc, characters)
```

## Server oldal

### Karakter létrehozás

A karakter létrehozás a `Player:CreateCharacter` metódust használja callback-en keresztül:

```lua
LS.Callback:Register('Identity:Server:CreateCharacter', function(source, data)
    local player = LS:GetPlayer(source)
    if not player then return false end

    local success = player:CreateCharacter(data)
    return success
end)
```

### CreateCharacter adat struktúra

```lua
{
    first_name = string,        -- Keresztnév
    middle_name = string,       -- Középső név (opcionális)
    last_name = string,         -- Vezetéknév
    birth_date = string,        -- Születési dátum
    gender = string,            -- Nem
    nationality = string,       -- Nemzetiség
    specialization = string,    -- Karakter út (citizen, criminal, stb.)
    appearance = table          -- Megjelenés adatok (ls_appearance formátum)
}
```

### GlobalState

```lua
GlobalState['FinishButtonEnabled'] = true  -- Befejezés gomb engedélyezése a létrehozásnál
```

## Kamera rendszer

A karakter választó saját kamera modult használ a háttér jelenet beállításához. A kamera a `Camera:Setup()` hívással inicializálódik és a karakter kiválasztása után automatikusan deaktiválódik.

## NUI kommunikáció

### Client → NUI

| Üzenet | Adat | Leírás |
|--------|------|--------|
| `setVisible` | `boolean` | NUI megjelenítés/elrejtés |
| `characterData` | `table` | Karakterek, slot-ok, specializációk |

### NUI → Client

| Callback | Adat | Leírás |
|----------|------|--------|
| `hideFrame` | — | NUI elrejtése és fókusz feloldása |

## Események

### Client

| Event | Paraméterek | Leírás |
|-------|-------------|--------|
| `Core:Client:PlayerJoined` | `characters` | Játékos belépett, karakterek betöltése |
| `Core:Client:OpenCharacterSelector` | `characters` | Karakter választó újranyitása |
| `Core:Client:CharacterLoaded` | `characterData` | Karakter sikeresen betöltve |

### Server Callbacks

| Callback | Paraméterek | Visszatérés | Leírás |
|----------|-------------|-------------|--------|
| `Identity:Server:CreateCharacter` | `data: table` | `boolean` | Új karakter létrehozása |

## Karakter limitek

- Alapértelmezetten maximum **3** karakter játékosonként (a `Player:CreateCharacter` érvényesíti)
- A slot konfiguráció admin csoportonként eltérhet
- Minden karakter egyedi 7 karakteres `stateId`-t kap (alfanumerikus)

## Példák

### Karakter választó kényszerített megnyitása

```lua
-- Szerver oldal: karakter választó újranyitása
local player = GetPlayer(source)
if not player then return end

local Database = require 'core.server.player.modules.database'
local characters = next(player.characters) and Database:GetCharacters(player.characters) or {}

TriggerClientEvent('Core:Client:OpenCharacterSelector', source, characters)
```

### Karakter adat lekérdezése választás után

```lua
-- Kliens oldal
RegisterNetEvent('Core:Client:CharacterLoaded', function(characterData)
    print('Karakter betöltve:', characterData.fullName)
    print('StateID:', characterData.stateId)
    print('Specializáció:', characterData.specialization)
end)
```
