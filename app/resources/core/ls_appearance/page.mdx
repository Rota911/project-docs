# Appearance

## Client oldali Exportok

## GetPedHairColor
Ez szerintem magától értetödő.

```lua
---@param ped number Ped entity
---@return table Hair color data (color, highlight)
local hairColor = exports.ls_appearance:GetPedHairColor(ped)
LS.Logger:Debug("Hair color data:", hairColor)
```

## GetPedHeadBlendData

Visszaadja a ped fejének a alakját egy táblában.

```lua
---@param ped number Ped entity
---@return table Head blend data (shapeFirst, shapeSecond, shapeThird, skinFirst, skinSecond, skinThird, shapeMix, skinMix, thirdMix, hasParent)
local headBlend = exports.ls_appearance:GetPedHeadBlendData(ped)
LS.Logger:Debug("Head blend data:", headBlend)
```

## GetPedHeadOverlay
Visszaadja a ped fejét.

```lua
---@param ped number Ped entity
---@return table Head overlay data, table Totals
local headOverlay = exports.ls_appearance:GetPedHeadOverlay(ped)
LS.Logger:Debug("Head overlay data:", headOverlay)
```

# Setters

## SetPedHeadBlendData

Beállítja a ped fejének az alakját.

```lua
---@param ped number Ped entity
---@param data table HeadBlend data (shapeFirst, shapeSecond, shapeThird, skinFirst, skinSecond, skinThird, shapeMix, skinMix, thirdMix, hasParent)
exports.ls_appearance:SetPedHeadBlendData(ped, data)
```

## SetPedHeadOverlay

Beállítja a ped fejének az overlay-jét (pl. szemszín, haj szín, szakáll, etc).

```lua
---@param ped number Ped entity
---@param data table HeadOverlay data (index, overlayValue, overlayOpacity, firstColor, secondColor, hairColor, hairHighlight)
exports.ls_appearance:SetPedHeadOverlay(ped, data)
```

## SetPedFaceFeature

Beállítja a ped arc jellegzetességeit (pl. orrforma, szája, stb).

```lua
---@param ped number Ped entity
---@param data table HeadStructure data (index, value)
exports.ls_appearance:SetPedFaceFeature(ped, data)
```

## SetPedDrawable

Beállítja a ped ruhadarabját.

```lua
---@param ped number Ped entity
---@param data table Drawable data (index, value, texture)
---@return number Total texture variations
exports.ls_appearance:SetPedDrawable(ped, data)
```

## SetPedProp

Beállítja a ped kiegészítőjét (pl. sapka, hátizsák, stb).

```lua
---@param ped number Ped entity
---@param data table Prop data (index, value, texture)
---@return number Total texture variations
exports.ls_appearance:SetPedProp(ped, data)
```

## SetPlayerModel

Beállítja a ped modelljét és opcionálisan a fejét és arc jellegzetességeit.

```lua
---@param ped number Ped entity
---@param data table Model data (model, headBlend, headStructure)
---@return number|boolean Returns ped or false on failure
exports.ls_appearance:SetPlayerModel(ped, data)
```

## SetPedHairColor

Beállítja a ped hajszínét.

```lua
---@param ped number Ped entity
---@param data table Hair color data (color, highlight)
exports.ls_appearance:SetPedHairColor(ped, data)
```

## SetPedClothes

Beállítja a ped teljes ruházatát (ruhadarabok, kiegészítők, arc overlay-k).

```lua
---@param ped number Ped entity
---@param data table Clothes data (drawables, props, headOverlay)
exports.ls_appearance:SetPedClothes(ped, data)
```

## SetPedSkin

Beállítja a ped teljes skinét (modell, fej alakja, arc jellegzetességei).

```lua
---@param ped number Ped entity
---@param data table Skin data (model, headBlend, headStructure)
---@return number|nil Returns ped or nil on failure
exports.ls_appearance:SetPedSkin(ped, data)
```

## SetPedAppearance

Beállítja a ped teljes megjelenését (skin, ruhadarabok, kiegészítők, hajszín).

```lua
---@param ped number Ped entity
---@param appearance table Full appearance data
---@return boolean Success
exports.ls_appearance:SetPedAppearance(ped, appearance)
```
