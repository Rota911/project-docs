# ls_appearance

Az **ls_appearance** egy modern és teljes körű karakter megjelenés szerkesztő resource FiveM szerverekhez.

## Áttekintés

Részletes karakter testreszabási lehetőségek web-based UI-val.

### Főbb funkciók

- 👤 **Ped modellek** - Férfi és női modellek támogatása
- 🎨 **Testreszabás** - Arc, haj, ruházat, tetovázások
- 💾 **Mentés/Betöltés** - Karakter megjelenés perzisztens tárolása
- 🏪 **Boltok** - Ruha és tetovázás üzletek
- 🎭 **Addon support** - Custom clothing és feature-ök
- 🔒 **Blacklist** - Tiltott clothing itemek

## API Export-ok

### Client

#### Open

Megjelenés szerkesztő megnyitása:

```lua
exports.ls_appearance:Open(options)
```

**Options:**
- `type` (string) - 'character', 'clothing', 'tattoo', 'barber'
- `ped` (number, optional) - Target ped (default: PlayerPedId())
- `allowModelChange` (boolean, optional) - Model váltás
- `allowSave` (boolean, optional) - Mentés engedélyezése
- `price` (number, optional) - Költség
- `onSave` (function, optional) - Callback mentéskor

#### Get Appearance

Jelenlegi ped megjelenésének lekérése:

```lua
local appearance = exports.ls_appearance:GetAppearance(ped)
```

#### Set Appearance

Megjelenés alkalmazása ped-re:

```lua
exports.ls_appearance:SetAppearance(ped, appearance)
```

#### Save Appearance

Megjelenés mentése database-be:

```lua
exports.ls_appearance:SaveAppearance(characterId, appearance)
```

#### Load Appearance

Megjelenés betöltése database-ből:

```lua
local appearance = exports.ls_appearance:LoadAppearance(characterId)
```

### Server

#### Get Appearance (Server)

```lua
local appearance = exports.ls_appearance:GetAppearance(characterId)
```

#### Save Appearance (Server)

```lua
exports.ls_appearance:SaveAppearance(characterId, appearance)
```

## Események

### Client események

```lua
-- Megjelenés szerkesztő megnyílt
AddEventHandler('ls_appearance:opened', function(type)
    print('Szerkesztő típus:', type)
end)

-- Megjelenés szerkesztő bezárult
AddEventHandler('ls_appearance:closed', function(saved)
    if saved then
        print('Változások mentve')
    else
        print('Megszakítva')
    end
end)

-- Megjelenés alkalmazva
AddEventHandler('ls_appearance:applied', function(appearance)
    print('Új megjelenés alkalmazva')
end)
```

### Server események

```lua
-- Megjelenés mentve
AddEventHandler('ls_appearance:saved', function(source, characterId, appearance)
    print(('Karakter %d megjelenése mentve'):format(characterId))
end)
```

### Uniform rendszer

```lua
-- Munkaruha felöltése
RegisterNetEvent('job:setUniform')
AddEventHandler('job:setUniform', function(uniformType)
    local uniforms = {
        police = {
            male = {
                components = {
                    {component = 3, drawable = 55, texture = 0},  -- Torso
                    {component = 4, drawable = 35, texture = 0},  -- Legs
                    {component = 6, drawable = 25, texture = 0},  -- Shoes
                    {component = 8, drawable = 58, texture = 0},  -- Undershirt
                    {component = 11, drawable = 55, texture = 0}  -- Top
                },
                props = {
                    {prop = 0, drawable = 46, texture = 0}  -- Hat
                }
            }
        }
    }
    
    local ped = PlayerPedId()
    local model = GetEntityModel(ped)
    local gender = model == GetHashKey('mp_m_freemode_01') and 'male' or 'female'
    
    local uniform = uniforms[uniformType][gender]
    
    if uniform then
        for _, component in ipairs(uniform.components) do
            SetPedComponentVariation(ped, component.component, component.drawable, component.texture, 0)
        end
        
        for _, prop in ipairs(uniform.props) do
            SetPedPropIndex(ped, prop.prop, prop.drawable, prop.texture, true)
        end
    end
end)
```

### Appearance visszaállítás

```lua
-- Civilian ruha visszaállítása (mentett megjelenés)
RegisterNetEvent('job:removeDuty')
AddEventHandler('job:removeDuty', function()
    local character = GetCharacter(source)
    
    if character and character.appearance then
        exports.ls_appearance:SetAppearance(PlayerPedId(), character.appearance)
    end
end)
```

## Troubleshooting

### Megjelenés nem töltődik be

```lua
-- Ellenőrizd a database kapcsolatot
local appearance = exports.ls_appearance:LoadAppearance(characterId)
if not appearance then
    print('Nincs mentett megjelenés')
    -- Használj default appearance-t
end
```

### Blacklist nem működik

```lua
-- Ellenőrizd a blacklist config-ot
local blacklist = require('shared.blacklist')
print(json.encode(blacklist))
```
