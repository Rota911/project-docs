# ls_voip

Proximity-alapú hang és rádió kommunikációs rendszer natív audio integrációval.

## Áttekintés

Az `ls_voip` kezeli a játékosok közötti hangkommunikációt, beleértve a proximity voice-ot (közelségi beszéd) és a rádió csatornákat. Natív audio fájlokat használ a rádió hangeffektekhez (PTT hangok).

## Függőségek

- `ls_core`
- `ls_radio` (kormányzati rádió csatorna konfiguráció)

## Konfiguráció

### FXManifest beállítások

A resource `use_experimental_fxv2_oal` flag-et használ a fejlett audio kezeléshez:

```lua
use_experimental_fxv2_oal 'yes'
```

### Natív audio

A resource saját audio bank-ot regisztrál a rádió hangeffektekhez:

```lua
data_file 'AUDIO_WAVEPACK'  'audiodirectory'
data_file 'AUDIO_SOUNDDATA' 'data/radioSfx.dat'
```

## Fájl struktúra

```
ls_voip/
  audiodirectory/
    radio.awc              -- Rádió hangeffektek (PTT be/ki)
  client/
    main.lua               -- Kliens oldali VOIP kezelés
    modules/
      *.lua                -- Kliens modulok
  data/
    radioSfx.dat54.rel     -- Natív audio definíció
  server/
    main.lua               -- Szerver oldali VOIP kezelés
  shared/
    main.lua               -- Közös beállítások
    configs/
      *.lua                -- Közös konfigurációk
```

## Főbb funkciók

- 🎤 **Proximity Voice** — Közelségi hangkommunikáció állítható hatótávval
- 📻 **Rádió csatornák** — Rádió kommunikáció az `ls_radio` integrációval
- 🔊 **Natív audio effektek** — PTT (Push-to-Talk) be/ki hangok
- 👥 **Kormányzati csatornák** — Előre definiált rádió csatornák az `ls_radio` konfigurációjából

## Közös beállítások

A `shared/main.lua` globális logger beállításokat definiál:

```lua
_G.LoggerSettings = {
    settings = {
        prefixes = { 'VOIP' },
        id = 'ls_voip'
    }
}
```

## Integráció az ls_radio-val

Az `ls_voip` az `ls_radio` resource kormányzati csatorna konfigurációját használja:

```lua
-- fxmanifest.lua
shared_scripts {
    '@ls_core/init.lua',
    'shared/main.lua',
    '@ls_radio/configs/govChannels.lua'
}
```

A kormányzati csatornák (pl. rendőrség, mentők) előre definiált rádió frekvenciákon érhetők el, amelyeket az `ls_radio` konfigurál.

## Jó tudni

- A VOIP rendszer az FXv2 OpenAL audio motort használja (`use_experimental_fxv2_oal`)
- A natív audio fájlok `.awc` formátumban vannak a rádió hangeffektekhez
- A proximity voice hatótáv kliens oldalon állítható
- A rádió csatornák szerver oldalon szinkronizálódnak
