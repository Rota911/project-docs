# ls_admin

Az admin modul lehetővé teszi parancsok és funkciók regisztrálását jogosultság ellenőrzéssel.

## RegisterFunction

Regisztrál egy szerver oldali funkciót jogosultság ellenőrzéssel.

```lua
---@param name string A funkció neve
---@param permissions string|string[]? Szükséges jogosultságok (alapértelmezett: 'user')
---@param callback fun(source: number, args: table): boolean, string? Végrehajtási funkció
---@param arguments Arguments[]? Argumentum definíciók
LS:RegisterFunction(name, permissions, callback, arguments)
```

### Példa

```lua
LS:RegisterFunction('teleport', 'admin', function(source, args)
    local targetPed = GetPlayerPed(args.playerId)
    local sourceCoords = GetEntityCoords(PlayerPedId())
    
    SetEntityCoords(targetPed, sourceCoords.x, sourceCoords.y, sourceCoords.z)
    return true, 'Teleportálva'
end, {
    { name = 'playerId', type = 'playerId', optional = false }
})
```

## Command

Regisztrál egy admin parancsot jogosultság ellenőrzéssel.

```lua
---@param command string|string[] Parancs neve vagy nevek
---@param permissions string|string[]? Szükséges jogosultságok (alapértelmezett: 'dev')
---@param callback fun(source: number, args: table, raw: string)|string Végrehajtási funkció vagy funkció név
---@param arguments Arguments[]? Argumentum definíciók
---@param helpText string? Súgó szöveg
LS:Command(command, permissions, callback, arguments, helpText)
```

### Példa - Direct callback

```lua
LS:Command('teleport', 'admin', function(source, args, raw)
    local targetPed = GetPlayerPed(args.playerId)
    local sourceCoords = GetEntityCoords(PlayerPedId())
    
    SetEntityCoords(targetPed, sourceCoords.x, sourceCoords.y, sourceCoords.z)
    TriggerClientEvent('chat:addMessage', source, {
        args = { 'ADMIN', 'Teleportálva!' }
    })
end, {
    { name = 'playerId', type = 'playerId', optional = false }
}, 'Teleportál egy játékost')
```

### Példa - RegisterFunction alapján

```lua
-- Először regisztrálj egy funkciót
LS:RegisterFunction('duty', 'user', function(source, args)
    local player = LS:GetPlayer(source)

    if not player then
        return false, 'Játékos nem található'
    end

    player:SetDuty()
    return true
end, nil)

-- Majd hozz létre egy parancsot, amely ezt a funkciót hívja
LS:Command('duty', 'user', 'duty', nil, 'Szolgálat be- és kikapcsolása.')
```

## ExecuteFunction

Végrehajtja a regisztrált funkciót jogosultság ellenőrzéssel.

```lua
---@param source number Játékos azonosító
---@param functionName string A funkció neve
---@param args table? Argumentumok
---@return boolean success Sikeres-e a végrehajtás
---@return string? message Hibaüzenet vagy eredmény
LS:ExecuteFunction(source, functionName, args)
```

### Példa

```lua
local success, message = LS:ExecuteFunction(source, 'duty', {})
if success then
    TriggerClientEvent('chat:addMessage', source, {
        args = { 'ADMIN', message }
    })
else
    TriggerClientEvent('chat:addMessage', source, {
        args = { 'HIBA', message }
    })
end
```

## Argumentum típusok

- `'string'` - Szöveg argumentum
- `'number'` - Szám argumentum
- `'playerId'` - Játékos azonosító (lehet 'me' vagy valós azonosító)
- `'any'` - Bármilyen típus
