# ls_admin

Az **ls_admin** egy teljes körű admin menü és eszköztár resource FiveM szerverekhez.

## Áttekintés

Admin eszközök és menürendszer szerver adminisztrátorok számára.

### Főbb funkciók

- 🎮 **Admin menü** - Grafikus felület admin műveletekhez
- 👥 **Játékos kezelés** - Teleport, freeze, kick, ban
- 🚗 **Jármű spawner** - Járművek létrehozása és kezelése
- 🎯 **Gyors eszközök** - Heal, armor, god mode
- 📍 **Teleport rendszer** - Koordináta és játékos alapú teleport
- ⚙️ **Szerver eszközök** - Időjárás, idő, bejelentések

## Telepítés

### Függőségek

```lua
-- ls_core framework szükséges
dependencies {
    'ls_core'
}
```

### server.cfg

```cfg
ensure ls_core
ensure ls_admin
```

## Használat

### Menü megnyitása

```lua
-- Kliens side
TriggerEvent('ls_admin:openMenu')

-- Alapértelmezett billentyű (általában F10 vagy hasonló)
```

### Permissions

Az admin jogosultságok az ls_core ACL rendszerén keresztül működnek:

```lua
-- Admin csoport beállítása
LS:AddPrincipal('player.1', 'admin')

-- Jogosultságok
owner       -- Teljes hozzáférés
dev         -- Fejlesztői eszközök
headadmin   -- Fő admin funkciók
admin       -- Alap admin eszközök
moderator   -- Moderátori eszközök
```

## Funkciók

### Játékos kezelés

#### Teleport játékoshoz

```lua
-- Példa parancs használat
/goto [playerId]  -- Teleport a játékoshoz
/bring [playerId] -- Játékos magadhoz hozása
```

#### Freeze/Unfreeze

```lua
/freeze [playerId]   -- Játékos lefagyasztása
/unfreeze [playerId] -- Feloldás
```

#### Kick/Ban

```lua
/kick [playerId] [reason]  -- Játékos kirúgása
-- Ban funkció menüből érhető el
```

### Jármű kezelés

#### Spawn Vehicle

```lua
-- Menüből: Vehicle Spawner
-- Kategóriák: Cars, Bikes, Helicopters, Boats, Emergency

-- Vagy parancs:
/car [modelName]  -- Jármű spawn
```

#### Jármű javítás

```lua
/fix    -- Aktuális jármű javítása
/dv     -- Aktuális jármű törlése
```

### Player eszközök

#### Heal & Armor

```lua
/heal [playerId]   -- Gyógyítás (opcionális target)
/armor [playerId]  -- Páncél adása
```

#### God Mode

```lua
-- Menüből: Player Options → God Mode

```

#### Invisible

```lua

```

## Integrációk

### ls_core integráció

```lua
-- Karakter alapú admin duty
local player = GetPlayer(source)
player:SetDuty(true)  -- Admin szolgálatba lépés

-- Admin státusz ellenőrzés
if player:GetDuty() then
    -- Admin szolgálatban
end
```

### Események

```lua
-- Kliens események
AddEventHandler('ls_admin:menuOpened', function()
    print('Admin menü megnyitva')
end)

AddEventHandler('ls_admin:menuClosed', function()
    print('Admin menü bezárva')
end)

-- Szerver események
AddEventHandler('ls_admin:playerAction', function(adminSource, targetSource, action)
    print(('Admin %d végrehajtotta: %s a játékoson %d'):format(
        adminSource, action, targetSource
    ))
end)
```

## Gyakorlati példák

### Custom admin action

```lua
-- Server side
RegisterNetEvent('ls_admin:customAction')
AddEventHandler('ls_admin:customAction', function(targetId, data)
    local source = source
    local player = GetPlayer(source)
    
    -- Admin check
    if not player or player.group ~= 'admin' then
        return
    end
    
    -- Művelet végrehajtása
    local target = GetPlayer(targetId)
    if target then
        -- ... custom logic
        
        -- Log
        exports.ls_admin:LogAction(source, 'custom_action', targetId)
    end
end)
```

### Custom menu item

```lua
-- Client side
RegisterNetEvent('ls_admin:addMenuItem')
AddEventHandler('ls_admin:addMenuItem', function()
    exports.ls_admin:AddMenuItem({
        label = 'Custom Function',
        icon = 'fas fa-star',
        permission = 'admin',
        action = function()
            TriggerServerEvent('myResource:customAction')
        end
    })
end)
```

### Admin log rendszer

```lua
-- Server side
AddEventHandler('ls_admin:actionLogged', function(adminId, action, targetId, data)
    local admin = GetPlayer(adminId)
    local target = targetId and GetPlayer(targetId)
    
    local logMessage = string.format(
        '[ADMIN] %s (%s) used %s',
        GetPlayerName(adminId),
        admin.group,
        action
    )
    
    if target then
        logMessage = logMessage .. string.format(' on %s', GetPlayerName(targetId))
    end
    
    print(logMessage)
    
    -- Database log
    MySQL.insert('INSERT INTO admin_logs (admin_id, action, target_id, data, timestamp) VALUES (?, ?, ?, ?, NOW())', {
        adminId, action, targetId, json.encode(data)
    })
end)
```

## Fejlesztés

### Új admin funkció hozzáadása

```lua
-- 1. Szerver funkció létrehozása
LS:RegisterFunction('myAdminFunction', 'admin', function(source, args)
    -- Admin funkció logika
    return true, 'Sikeres művelet'
end, {
    {name = 'target', type = 'playerId'}
})

-- 2. Kliens menü item
exports.ls_admin:AddMenuItem({
    category = 'Player Management',
    label = 'My Function',
    icon = 'fas fa-cog',
    permission = 'admin',
    action = function(targetId)
        LS:ExecuteFunction('myAdminFunction', {target = targetId})
    end
})
```

## Troubleshooting

### Menü nem nyílik meg

```lua
-- Ellenőrizd a jogosultságokat
local player = GetPlayer(source)
print('Player group:', player.group)

-- Ellenőrizd az ACL-t
print('Has admin ace:', LS:IsPlayerAceAllowed(source, 'admin'))
```

### Noclip nem működik

```lua
-- Kliens oldalon
-- Győződj meg róla, hogy a noclip engedélyezve van
SetPlayerInvincible(PlayerId(), true)
FreezeEntityPosition(PlayerPedId(), true)
```
