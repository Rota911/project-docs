## Logger:Debug()
Fejlesztői (debug) üzenetet ír ki, csak fejlesztői környezetben.
- module: `string` – Modul neve
- message: `string` – Üzenet szövege
- ...: `any` – További formázási paraméterek

```lua copy
Logger:Debug('UI', 'Teszt debug üzenet: %s', 'adat')
```

## Logger:Info()
Információs üzenetet ír ki.
- module: `string` – Modul neve
- message: `string` – Üzenet szövege
- ...: `any` – További formázási paraméterek

```lua copy
Logger:Info('UI', 'Panel megnyitva: %s', panelName)
```

## Logger:Warn()
Figyelmeztető üzenetet ír ki.
- module: `string` – Modul neve
- message: `string` – Üzenet szövege
- ...: `any` – További formázási paraméterek

```lua copy
Logger:Warn('UI', 'Figyelmeztetés: %s', 'Hibás panel')
```

## Logger:Error()
Hibaüzenetet ír ki.
- module: `string` – Modul neve
- message: `string` – Üzenet szövege
- ...: `any` – További formázási paraméterek

```lua copy
Logger:Error('UI', 'Hiba történt: %s', 'Panel nem található')
```

## Események
Kliens oldali események naplózáshoz:
- `Logger:Debug`
- `Logger:Info`
- `Logger:Warn`
- `Logger:Error`

```lua copy
TriggerEvent('Logger:Info', 'UI', 'Panel megnyitva: %s', panelName)
```
