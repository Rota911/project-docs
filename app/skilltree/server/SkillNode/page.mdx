# SkillNode
* id: `string` – Node ID
* data: `table` – Node configuration
  * preRequsites: `table<string>` – Nodes required before unlocking
  * children: `table<string>` – Child node IDs
  * lock: `table` – Lock conditions
  * unlocked: `boolean` – Default unlock state
  * label: `string` – Display name
  * description: `string` – Description text
  * cost: `number` – Required points to unlock

---

## SkillNode:Unlock(charId)
Attempts to unlock the node for the given character.

- charId: `number|string` – Character identifier

**Return:**  
- `boolean` – Success  
- `string|nil` – Error reason (if any)

```lua copy
local node = tree:GetNode('warrior_strength')

local ok, reason = node:Unlock(1)
if ok then
    print('Node unlocked!')
else
    print('Failed to unlock:', reason)
end
```

## SkillNode:IsUnlocked(charId)

Checks whether this node is already unlocked by the character.

* charId: `number|string` – Character identifier

**Return:**

* `boolean` – Whether the node is unlocked

```lua copy
local node = tree:GetNode('warrior')

if node:IsUnlocked(charId) then
    print('Warrior node unlocked.')
end
```
## SkillNode:GetChildNodes()

Returns all children of this node as SkillNode objects.

**Return:**

* `table<string, SkillNode>` – List of child nodes

```lua copy
local node = tree:GetNode('warrior')
local children = node:GetChildNodes()

for _, child in ipairs(children) do
    print('Child node:', child.id)
end
```
## SkillNode:GetParentNode()

Finds and returns the parent node of this node.

**Return:**

* `SkillNode|nil` – Parent node, or `nil` if this is a root node

```lua copy
local node = tree:GetNode('warrior_strength')
local parent = node:GetParentNode()

print('Parent node:', parent and parent.id or 'None')
```

## SkillNode:GetAllDescendants()

Returns all descendant nodes (children, grandchildren, etc.).

**Return:**

* `SkillNode[]` – List of all descendant nodes

```lua copy
local node = tree:GetNode('warrior')
local descendants = root:GetAllDescendants()

print('Descendants:')
for _, n in ipairs(descendants) do
    print(' -', n.id)
end
```

## SkillNode:GetRootNode()

Returns the top-most ancestor node in the tree.

**Return:**

* `SkillNode` – Root node

```lua copy
local node = tree:GetNode('warrior_strength_advanced')
local root = node:GetRootNode()

print('Root node:', root.name) -- likely "warrior"
```
