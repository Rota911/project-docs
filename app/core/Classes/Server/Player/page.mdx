## PlayerClass

A `PlayerClass` egy játékos alapvető adatait és funkcióit kezeli. Minden játékos rendelkezik egy player objektummal, amely a karakterválasztástól kezdve elérhető.

### Tulajdonságok

- `source: number` – A játékos forrásazonosítója (server ID)
- `identifier: string` – License2 azonosító (pl.: `98399a05dac2ccae820142f8af6804178b5f178a`)
- `discord: string` – Discord azonosító (pl.: `212704429323976706`)
- `playerId: number` – Adatbázisban tárolt játékos azonosító (pl.: `10001`)
- `isLoggedIn: boolean` – Be van-e jelentkezve karakterbe
- `adminGroup: string` – Admin csoport neve (alapértelmezett: `user`)
- `characters: table` – A játékos karakter azonosítóinak listája
- `points: number` – Prémium pontok száma
- `duty: boolean` – Admin szolgálatban van-e

## PlayerClass:Event
Eseményt küld a kliensnek.

**Paraméterek**
- eventName: `string` – Az esemény neve
- ...: `any` – További paraméterek, amelyeket az eseménynek átadunk

```lua copy
player:Event('LS:Client:Clipboard', 'Tevepata')
player:Event('LS:Client:PlayerDataChanged', 'points', 100)
```

## PlayerClass:SetAsLoggedIn
Beállítja a beléptetés státuszát.

**Paraméterek**
- state: `boolean` – Be van-e jelentkezve karakterbe

**Visszatérési érték**
- nincs

```lua copy
player:SetAsLoggedIn(true)
```

## PlayerClass:SetGroup
Megváltoztatja az admin csoportot.

**Paraméterek**
- group: `string` – Új admin csoport (`moderator` | `admin` | `headadmin` | `dev` | `owner`)
- temporary?: `boolean` – Ideiglenes, restartig tart

**Visszatérési érték**
- `boolean, string` – Sikeresség és üzenet

1. példa: Csoport beállítása véglegesen
```lua copy
local success, msg = player:SetGroup('admin')
print(success, msg)
```

2. példa: Csoport beállítása ideiglenesen
```lua copy
player:SetGroup('headadmin', true)
```

## PlayerClass:SetDuty
Admin szolgálatba lépteti a játékost.

**Paraméterek**
- state?: `boolean` – Szolgálatba léptetés (`true`), vagy kiléptetés (`false`). Ha nincs megadva, átváltja az aktuális állapotot.

**Visszatérési érték**
- nincs

1. példa: Szolgálatba léptetés
```lua copy
player:SetDuty(true)
```

2. példa: Kiléptetés szolgálatból
```lua copy
player:SetDuty(false)
```

3. példa: Állapot átváltása
```lua copy
player:SetDuty()
```

## PlayerClass:GetDuty
Visszaadja, hogy a játékos admin szolgálatban van-e.

**Visszatérési érték**
- `boolean` – Szolgálatban van-e

```lua copy
local isDuty = player:GetDuty()
print(isDuty)
```

## PlayerClass:CreateCharacter
Létrehoz egy új karaktert a játékoshoz.

**Paraméterek**
- `characterData: table` – Karakter adatok
  - `first_name: string` – Keresztnév
  - `last_name: string` – Vezetéknév
  - `birth_date: string` – Születési dátum (formátum: `YYYY-MM-DD`)
  - `gender: string` – Nem (`male` | `female`)
  - `nationality: string` – Nemzetiség
  - `pathType: string` – Karakter út típusa (pl.: `citizen`, `criminal`)

**Visszatérési érték**
- `boolean, string` – Sikeresség és üzenet
  - `true, 'success_create_character'` – Sikeres létrehozás
  - `false, 'maximum_character'` – Elérte a maximum karakterszámot (3)
  - `false, 'cannot_save_character'` – Nem sikerült menteni

```lua copy
local success, msg = player:CreateCharacter({
    first_name = 'John',
    last_name = 'Doe',
    birth_date = '1990-01-01',
    gender = 'male',
    nationality = 'Hungarian',
    pathType = 'citizen'
})
print(success, msg)
```

## PlayerClass:SetActiveCharacter
Beállítja és betölti a játékos aktív karakterét.

**Paraméterek**
- `characterId: number` – Karakter azonosító

**Visszatérési érték**
- `boolean` – Sikeres beállítás

**Megjegyzés**
- Csak akkor használható, ha a játékos még nincs bejelentkezve karakterbe
- Automatikusan létrehozza a karakter objektumot és triggereli a betöltési eventeket

```lua copy
local success = player:SetActiveCharacter(42)
if success then
    print('Karakter sikeresen betöltve')
end
```
## PlayerClass:GetActiveCharacter
Visszaadja a játékos aktív karakterét.

**Visszatérési érték**
- `Character|nil` – A karakter objektum, vagy `nil` ha nincs bejelentkezve

```lua copy
local character = player:GetActiveCharacter()
if character then
    print('Karakter neve:', character.fullName)
end
```
