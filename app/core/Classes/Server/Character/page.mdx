## CharacterClass

- source: `number` - Játékos source
- charId: `number` - Karakter azonosító, Pl: `42`
- playerId: `number` - Játékos azonosító, Pl: `10001`
- stateId: `string` - Állapot azonosító, Pl: `0OW2OX2`
- firstName: `string` - Keresztnév
- lastName: `string` - Vezetéknév
- fullName: `string` - Teljes név
- dateOfBirth: `string` - Születési dátum
- nationality: `string` - Nemzetiség
- gender: `string` - Nem
- groups: `table` - Csoportok, amelyekhez tartozik
- pathType: `string` - Kiválasztott életút
- position: `vector4` - Jelenlegi pozíció (5 percenként frissül, valamint kilépéskor.)
- statuses: `table` - Státuszok

## CharacterClass:Event
Eseményt küld a kliens számára.

**Paraméterek**
- eventName: `string` – Az esemény neve
- ...: `any` – További paraméterek, amelyeket az eseménynek átadunk

```lua copy
character:Event('Core:Client:Clipboard', 'Tevepata')
```

## CharacterClass:Save
Elmenti a karakter adatait az adatbázisba.

**Paraméterek**
- logout?: `boolean` – Ha igaz, akkor a jelenlegi pozíciót menti, egyébként a tárolt pozíciót

**Visszatérési érték**
- `boolean` – `true` ha sikeres a mentés, `false` ha sikertelen

```lua copy
local success = character:Save(true)
if success then
    print('Karakter sikeresen mentve')
else
    print('Hiba a karakter mentésekor')
end
```

## CharacterClass:Unload
Kijelentkezteti a karaktert.

**Paraméterek**
- instant?: `boolean` – Ha igaz, azonnali kiléptetés callback nélkül

**Visszatérési érték**
- `boolean` – `true` ha sikeres a kiléptetés, `false` ha sikertelen

1. példa: Normál kiléptetés
```lua copy
local success = character:Unload()
if success then
    print('Karakter sikeresen kijelentkeztetve')
end
```

2. példa: Azonnali kiléptetés
```lua copy
character:Unload(true) -- Nincs callback, azonnal kilép
```

## CharacterClass:SetData
Adatot ad hozzá vagy módosít a karakteren.

**Paraméterek**
- key: `string` – Kulcs
- value: `any` – Érték
- state?: `boolean` – Legyen-e statebag-re rakva?
- replicated?: `boolean` – Legyen-e replikálva kliens oldalra a state?

1. példa: Egyszerű adat beállítás
```lua copy
character:SetData('customData', { value = 100 })
```

2. példa: Adat beállítás statebag-gel
```lua copy
character:SetData('inventory', items, true, false)
```

3. példa: Adat beállítás kliens replikációval
```lua copy
character:SetData('health', 100, true, true)
```

## CharacterClass:GetData
Adatot kér le a karakterről.

**Paraméterek**
- key: `string` – Kulcs

**Visszatérési érték**
- `any` – A kért kulcshoz tartozó érték

```lua copy
local firstName = character:GetData('firstName')
local customData = character:GetData('customData')
local position = character:GetData('position')
```

## CharacterClass:SetCoords
A karaktert megadott koordinátákra teleportálja.

**Paraméterek**
- newCoords: `vector4|vector3|table` – Az új koordináták

1. példa: Teleportálás vector4-gyel
```lua copy
character:SetCoords(vector4(100.0, 200.0, 30.0, 90.0))
```

2. példa: Teleportálás vector3-mal (heading nélkül)
```lua copy
character:SetCoords(vector3(100.0, 200.0, 30.0))
```

3. példa: Teleportálás tábla formátumban
```lua copy
character:SetCoords({ x = 100.0, y = 200.0, z = 30.0, heading = 90.0 })
```

## CharacterClass:GetCoords
Visszaadja a karakter jelenlegi pozícióját.

**Paraméterek**
- _type?: `"vector4"|"vector3"|"vector2"|"table"` – Visszatérési érték típusa (alapértelmezett: `"vector4"`)

**Visszatérési érték**
- `vector4|vector3|vector2|table` – A karakter koordinátái a kért formátumban

1. példa: Koordináták lekérése vector4-ként (alapértelmezett)
```lua copy
local coords = character:GetCoords()
print(coords.x, coords.y, coords.z, coords.w)
```

2. példa: Koordináták lekérése vector3-ként
```lua copy
local coords = character:GetCoords('vector3')
print(coords.x, coords.y, coords.z)
```

3. példa: Koordináták lekérése táblázatként
```lua copy
local coords = character:GetCoords('table')
print(coords.x, coords.y, coords.z, coords.heading)
```

## CharacterClass:SetGroup
Karaktert hozzáadja egy csoporthoz vagy eltávolítja onnan. Ez a metódus átirányít a GroupClass metódusokhoz.

**Paraméterek**
- group: `string` – Csoport azonosító
- rank?: `string` – Rang azonosító (nil esetén eltávolítás a csoportból)
- silent?: `boolean` – Ne triggerelj eventeket

**Visszatérési érték**
- `boolean, string?` – Sikeresség és esetleges hibaüzenet

**Példa 1:** Karakter hozzáadása csoporthoz
```lua copy
local success, msg = character:SetGroup('lspd', 'officer')
if success then
    print('Karakter hozzáadva a csoporthoz')
else
    print('Hiba:', msg)
end
```

**Példa 2:** Karakter eltávolítása csoportból
```lua copy
character:SetGroup('lspd') -- rang nélkül = eltávolítás
```

**Példa 3:** Csendesen hozzáadás (nincs értesítés)
```lua copy
character:SetGroup('lspd', 'default', true)
```

## CharacterClass:SetGroupInternal
Karaktert hozzáadja egy csoporthoz vagy eltávolítja onnan közvetlenül, validációk nélkül. **Ezt a metódust csak a GroupClass használja, ne hívd meg kívülről!**

**Paraméterek**
- group: `string` – Csoport azonosító
- rank?: `string` – Rang azonosító (nil esetén eltávolítás)
- silent?: `boolean` – Ne triggerelj eventeket

**Visszatérési érték**
- `boolean, string?` – Sikeresség és esetleges hibaüzenet

## CharacterClass:HasGroup
Ellenőrzi, hogy a karakter tagja-e egy csoportnak.

**Paraméterek**
- group: `string` – Csoport azonosító
- rank?: `string` – Konkrét rang ellenőrzése (opcionális)

**Visszatérési érték**
- `boolean` – Tagja-e a csoportnak (és adott rangnak)

**Példa 1:** Csoporttagság ellenőrzése
```lua copy
if character:HasGroup('lspd') then
    print('A karakter LSPD tag')
end
```

**Példa 2:** Konkrét rang ellenőrzése
```lua copy
if character:HasGroup('lspd', 'boss') then
    print('A karakter LSPD főnök')
end
```
