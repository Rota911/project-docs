# Interaction

Client-side interaction system for zones, entities, models, and global options. Access via `LS.Interact`.

> Requires `ls_interaction` to be started.

## Zone Interactions

### AddPolyZone

```lua
---@param data ZoneOption
---@return number zoneId
LS.Interact:AddPolyZone(data)
```

### AddBoxZone

```lua
---@param data ZoneOption
---@return number zoneId
LS.Interact:AddBoxZone(data)
```

### AddSphereZone

```lua
---@param data ZoneOption
---@return number zoneId
LS.Interact:AddSphereZone(data)
```

### ZoneExists

```lua
---@param zoneId number|string
---@return boolean
LS.Interact:ZoneExists(zoneId)
```

### RemoveZone

```lua
---@param zoneId number|string
---@param suppressWarning? boolean
LS.Interact:RemoveZone(zoneId, suppressWarning)
```

## Global Options

Add interaction options that appear on all entities of a given type.

### Ped

```lua
---@param options InteractionOption|InteractionOption[]
---@param optionType? 'target'|'interaction'
```

```lua
---@param names string|string[]
LS.Interact:RemoveGlobalPed(names)
```

## Model Options

```lua
---@param models string|number|(string|number)[]
---@param options InteractionOption|InteractionOption[]
---@param optionType? 'target'|'interaction'
LS.Interact:AddModel(models, options, optionType)
```

```lua
---@param models string|number|(string|number)[]
---@param names? string|string[] -- nil removes all options
LS.Interact:RemoveModel(models, names)
```
