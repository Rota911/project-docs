# Blips

Client-side map blip manager with category toggling. Access via `LS.Blips`.

## Create

```lua
---@param data { position: vector3, settings?: BlipSettings }
---@return Blip|nil
LS.Blips:Create(data)
```

## Remove

```lua
---@param blipId number
---@return boolean
LS.Blips:Remove(blipId)
```

## Get

```lua
---@param blipId number
---@return Blip|nil
LS.Blips:Get(blipId)
```

## ToggleCategory

```lua
---@param category string
---@param hide boolean
LS.Blips:ToggleCategory(category, hide)
```

Hides or shows all blips with the given category. New blips created in a hidden category are auto-hidden.

## GetCategories

```lua
---@return table<string, boolean>
LS.Blips:GetCategories()
```

Returns a set of all category names currently in use.

## Blip Instance

Returned by `Create`. Each blip has a unique numeric `id` and a native `handle`.

### Show / Hide

```lua
blip:Show()
blip:Hide()
```

`Show` is blocked if the blip's category is currently hidden.

### SetWayPoint

```lua
blip:SetWayPoint()
```

### SetRoute

```lua
---@param color? number
blip:SetRoute(color)
```

### UpdateLocation

```lua
---@param position vector2|vector3
blip:UpdateLocation(position)
```

### Destroy

```lua
blip:Destroy()
```

Removes the native blip and unregisters it from the manager.

## BlipSettings

```lua
---@class BlipSettings
---@field sprite? number
---@field color? number
---@field short? boolean
---@field scale? number
---@field heading? boolean
---@field category? string|number
---@field tick? boolean
---@field text? string
```

## Events

| Event | Parameters | Description |
|-------|-----------|-------------|
| `Core:Client:ToggleBlipCategory` | `category: string, hide: boolean` | Toggles category visibility |
