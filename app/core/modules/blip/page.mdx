# Blip modul

A Blip modul lehetővé teszi térképi blipek létrehozását, kezelését és testreszabását.

## Funkciók

### Create

Új blip létrehozása a térképen.

```lua
local blip = LS.Blips:Create({
    position = vector3(x, y, z),
    settings = {
        sprite = 1,
        color = 1,
        text = 'Blip név',
        scale = 1.0,
        short = true,
        category = 'shops',
        heading = false,
        tick = false
    }
})
```

**Paraméterek:**
- `data` (table): Blip adatok
  - `position` (vector3): A blip pozíciója
  - `settings` (table, opcionális): Blip beállítások
    - `sprite` (number): Blip sprite ID
    - `color` (number): Blip szín ID
    - `text` (string): Blip neve
    - `scale` (number): Blip mérete (alapértelmezett: 1.0)
    - `short` (boolean): Csak közeli távolságban látható
    - `category` (string): Kategória neve
    - `heading` (boolean): Irány jelző megjelenítése
    - `tick` (boolean): Pipa ikon megjelenítése

**Visszatérési érték:**
- `Blip`: A blip objektum

**Példák:**

```lua
-- Egyszerű blip
local myBlip = LS.Blips:Create({
    position = vector3(0.0, 0.0, 0.0),
    settings = {
        sprite = 1,
        color = 2,
        text = 'Fontos hely'
    }
})

-- Bolt blip kategóriával
local shopBlip = LS.Blips:Create({
    position = vector3(25.7, -1347.3, 29.49),
    settings = {
        sprite = 52,
        color = 2,
        text = '24/7 Bolt',
        scale = 0.8,
        short = true,
        category = 'shops'
    }
})

-- Garázs blip iránymutató jellel
local garageBlip = LS.Blips:Create({
    position = vector3(215.0, -810.0, 30.7),
    settings = {
        sprite = 50,
        color = 3,
        text = 'Garázs',
        heading = true
    }
})
```

### Remove

Blip törlése azonosító alapján.

```lua
local success = LS.Blips:Remove(blipId)
```

**Paraméterek:**
- `blipId` (number): A blip egyedi azonosítója

**Visszatérési érték:**
- `boolean`: Sikeres volt-e a törlés

**Példa:**

```lua
local blip = LS.Blips:Create({
    position = vector3(0.0, 0.0, 0.0),
    settings = { sprite = 1, text = 'Teszt' }
})

-- Később törlés
LS.Blips:Remove(blip.id)
```

### Get

Blip lekérése azonosító alapján.

```lua
local blip = LS.Blips:Get(blipId)
```

**Paraméterek:**
- `blipId` (number): A blip egyedi azonosítója

**Visszatérési érték:**
- `Blip|nil`: A blip objektum vagy nil ha nem található

**Példa:**

```lua
local blip = LS.Blips:Get(123456)
if blip then
    print('Blip megtalálva:', blip.settings.text)
end
```

### ToggleCategory

Teljes kategória elrejtése vagy megjelenítése.

```lua
LS.Blips:ToggleCategory(category, hide)
```

**Paraméterek:**
- `category` (string): A kategória neve
- `hide` (boolean): true = elrejt, false = megjelenít

**Példa:**

```lua
-- Összes 'shops' kategóriájú blip elrejtése
LS.Blips:ToggleCategory('shops', true)

-- Összes 'jobs' kategóriájú blip megjelenítése
LS.Blips:ToggleCategory('jobs', false)
```

### GetCategories

Az összes létező kategória lekérése.

```lua
local categories = LS.Blips:GetCategories()
```

**Visszatérési érték:**
- `table`: Kategóriák listája (key = kategória név, value = true)

**Példa:**

```lua
local categories = LS.Blips:GetCategories()
for category, _ in pairs(categories) do
    print('Kategória:', category)
end
```

## Blip objektum metódusok

Miután létrehoztál egy blipet, a következő metódusok érhetők el rajta:

### Show

Blip megjelenítése a térképen.

```lua
blip:Show()
```

**Példa:**

```lua
local blip = LS.Blips:Create({
    position = vector3(0.0, 0.0, 0.0),
    settings = { sprite = 1 }
})

blip:Hide()  -- Elrejtés
Wait(5000)
blip:Show()  -- Újra megjelenítés
```

### Hide

Blip elrejtése a térképről.

```lua
blip:Hide()
```

**Példa:**

```lua
blip:Hide()
```

### SetWayPoint

Waypoint (navigációs cél) beállítása a blip pozíciójára.

```lua
blip:SetWayPoint()
```

**Példa:**

```lua
local blip = LS.Blips:Create({
    position = vector3(100.0, 200.0, 30.0),
    settings = { sprite = 1, text = 'Célpont' }
})

-- GPS útvonal beállítása
blip:SetWayPoint()
```

### SetRoute

Útvonal rajzolása a bliphez.

```lua
blip:SetRoute(color)
```

**Paraméterek:**
- `color` (number, opcionális): Az útvonal színe

**Példa:**

```lua
-- Alapértelmezett színű útvonal
blip:SetRoute()

-- Piros színű útvonal
blip:SetRoute(6)
```

### UpdateLocation

Blip pozíciójának frissítése.

```lua
blip:UpdateLocation(position)
```

**Paraméterek:**
- `position` (vector2|vector3): Az új pozíció

**Példa:**

```lua
local blip = LS.Blips:Create({
    position = vector3(0.0, 0.0, 0.0),
    settings = { sprite = 1, text = 'Mozgó cél' }
})

-- Pozíció frissítése
CreateThread(function()
    while true do
        Wait(1000)
        local newPos = GetEntityCoords(someEntity)
        blip:UpdateLocation(newPos)
    end
end)
```

### Destroy

Blip törlése.

```lua
blip:Destroy()
```

**Példa:**

```lua
blip:Destroy()
```

## Events

### Core:Client:ToggleBlipCategory

Kategória elrejtése/megjelenítése event-en keresztül.

```lua
TriggerEvent('Core:Client:ToggleBlipCategory', 'shops', true)
```

**Példa:**

```lua
-- Összes bolt blip elrejtése
TriggerEvent('Core:Client:ToggleBlipCategory', 'shops', true)

-- Összes munka blip megjelenítése
TriggerEvent('Core:Client:ToggleBlipCategory', 'jobs', false)
```

## Gyakorlati példák

### Dinamikus játékos követő blip

```lua
local playerBlips = {}

-- Játékos blip létrehozása
RegisterNetEvent('createPlayerBlip', function(playerId, playerName)
    local targetPed = GetPlayerPed(GetPlayerFromServerId(playerId))
    
    playerBlips[playerId] = LS.Blips:Create({
        position = GetEntityCoords(targetPed),
        settings = {
            sprite = 1,
            color = 4,
            text = playerName,
            short = false
        }
    })
    
    -- Pozíció frissítés thread
    CreateThread(function()
        while playerBlips[playerId] do
            Wait(500)
            if DoesEntityExist(targetPed) then
                playerBlips[playerId]:UpdateLocation(GetEntityCoords(targetPed))
            end
        end
    end)
end)

-- Játékos blip törlése
RegisterNetEvent('removePlayerBlip', function(playerId)
    if playerBlips[playerId] then
        playerBlips[playerId]:Destroy()
        playerBlips[playerId] = nil
    end
end)
```

### Bolt rendszer blipekkel

```lua
local shops = {
    {pos = vector3(25.7, -1347.3, 29.49), name = '24/7 Supermarket'},
    {pos = vector3(-3038.93, 585.95, 7.9), name = '24/7 Supermarket'},
    {pos = vector3(-3241.93, 1001.46, 12.83), name = '24/7 Supermarket'},
}

-- Blipek létrehozása
for i, shop in ipairs(shops) do
    LS.Blips:Create({
        position = shop.pos,
        settings = {
            sprite = 52,
            color = 2,
            text = shop.name,
            scale = 0.7,
            short = true,
            category = 'shops'
        }
    })
end

-- Blipek ki/be kapcsolása paranccsal
RegisterCommand('toggleshops', function()
    local categories = LS.Blips:GetCategories()
    if categories['shops'] then
        -- Toggle logic
        LS.Blips:ToggleCategory('shops', not hiddenShops)
        hiddenShops = not hiddenShops
    end
end)
```

### Misszió blip útvonallal

```lua
local missionBlip = LS.Blips:Create({
    position = vector3(1000.0, 2000.0, 30.0),
    settings = {
        sprite = 566,
        color = 5,
        text = 'Misszió célpont',
        scale = 1.2,
        tick = true
    }
})

-- Útvonal beállítása pirosra
missionBlip:SetRoute(6)

-- Misszió befejezése után törlés
RegisterNetEvent('missionComplete', function()
    missionBlip:Destroy()
end)
```

## Jó tudni

- A blipek automatikusan kapnak egyedi ID-t (111111-999999 között)
- A kategóriák segítségével csoportosíthatod és egyszerre kezelheted a blipeket
- Az elrejtett kategóriába tartozó blipek automatikusan elrejtve jönnek létre
- A blip objektumok referenciát tárolnak, így bármikor módosíthatók
- A `short` beállítás hasznos kisebb, helyi célpontokhoz
