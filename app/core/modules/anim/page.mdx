# Animation

Client-side animation and scenario system with prop sync and presets. Access via `LS.Anim`.

## Play

```lua
---@param ped number
---@param animDictionary string
---@param animationName? string
---@param options? AnimPlayOptions
---@return boolean success
LS.Anim:Play(ped, animDictionary, animationName, options)
```

If `animDictionary` matches a preset name, the preset's dict/name/options are used instead.

```lua
---@class AnimPlayOptions
---@field reverse? boolean
---@field blendInSpeed? number -- default 8.0
---@field blendOutSpeed? number -- default -8.0
---@field duration? integer -- default -1 (infinite)
---@field animFlags? AnimationFlags -- default 0 (auto-set to 1 if duration is -1)
---@field startPhase? number -- default 0.0
---@field phaseControlled? boolean
---@field controlledPhase? boolean
---@field overrideCloneUpdate? boolean
---@field prop? AnimPropOptions | AnimPropOptions[]
---@field playOnce? boolean If true, the animation will be removed from the ped once it finishes playing. Defaults to false.
```

```lua
---@class AnimPropOptions
---@field model? number
---@field bone? number
---@field offset? vector3
---@field rot? vector3
---@field rotationOrder? number -- default 2
```

Props are synced to other players via `Core:Server:AnimProps`.

## GetPlaying

Returns the current animation state stored on the ped's `StateBag`, or `nil`.

```lua
---@param ped number
---@return { dict: string, name: string, options: AnimPlayOptions, scenario: string }?
LS.Anim:GetPlaying(ped)
```

## Stop

```lua
---@param ped number
---@param options? { blendOutSpeed?: number }
LS.Anim:Stop(ped, options)
```

Clears state, stops the anim task or scenario, and removes synced props.

## AnimationFlags

Combinable via bitwise OR.

| Value | Name |
|-------|------|
| 0 | DEFAULT |
| 1 | LOOPING |
| 2 | HOLD_LAST_FRAME |
| 8 | NOT_INTERRUPTABLE |
| 16 | UPPERBODY |
| 32 | SECONDARY |
| 128 | ABORT_ON_PED_MOVEMENT |
| 1048576 | HIDE_WEAPON |

Common combos: `49` = LOOPING \| UPPERBODY \| SECONDARY, `9` = LOOPING \| NOT_INTERRUPTABLE.

## Presets

Pass the preset key as `animDictionary` to `Play`. Dict, name, and options are resolved automatically.

| Key | Description |
|-----|-------------|
| `phone` | Phone texting with `prop_player_phone_01` |
| `broom` | Sweeping with `prop_tool_broom` |
| `mop` | Mopping with `prop_cs_mop_s` |
| `rake` | Raking with `prop_tool_rake` |
| `dig` | Digging with `prop_tool_shovel` |
| `leafblower` | Leaf blower with `prop_leaf_blower_01` |
| `armchair` | Sitting in armchair with `p_armchair_01_s` |
| `radioTalk` | Walkie-talkie talk with `prop_cs_hand_radio` |
| `radioHold` | Holding radio with `ls_radio1` |
| `megaphone` | Megaphone with `prop_megaphone_01` |

```lua
LS.Anim:Play(cache.ped, 'phone')
```

## Auto-Recovery

Animations auto-restart after:
- `CEventPlayerCollisionWithPed` — restarts after 500ms debounce
- `CEventOpenDoor` — restarts after door animation finishes

Both are suppressed while `ls_ui:IsProgressbarActive()` returns true.
