# Camera

Client-side scripted camera system with transitions, follow, orbit, pan, dolly, and mouse control. Access via `LS.Camera`.

## Create

```lua
---@param data? CameraConstructorData
---@return Camera
LS.Camera:Create(data)
```

```lua
---@class CameraConstructorData
---@field attach? CameraConstructorAttach
---@field coords? vector3
---@field rotation? vector3
---@field fov? number
---@field activate? boolean
---@field activateDelay? number
```

```lua
---@class CameraConstructorAttach
---@field entity number
---@field offset? vector3
---@field isRelative? boolean
---@field point? boolean -- if true, camera points at the entity
```

```lua
local cam = LS.Camera:Create({
    coords = vector3(0.0, 0.0, 50.0),
    rotation = vector3(-30.0, 0.0, 0.0),
    fov = 60.0,
    activate = true,
})
```

## Camera Instance

All methods below are called on the camera object returned by `Create`.

### Lifecycle

```lua
---@param delay? number -- ms before activating
cam:Activate(delay)
```

```lua
cam:Deactivate()
```

```lua
---@return boolean
cam:IsActive()
```

```lua
cam:Destroy()
```

`Destroy` stops all running threads (follow, orbit, pan, dolly, shake, mouse control, transition), deactivates the camera, and renders gameplay cam.



### LookAt

Points the camera at a coordinate or entity. Optionally interpolates over `duration`.

```lua
---@param target vector3|number -- coords or entity
---@param duration? number -- ms, instant if nil
cam:LookAt(target, duration)
```

### Freeze

```lua
---@param state boolean
cam:Freeze(state)
```

### Transition

```lua
---@param data { coords?: vector3, rotation?: vector3, fov?: number, duration: number, easing?: string }
cam:TransitionTo(data)
```

Smoothly transitions camera properties over `duration` ms.

### InterpolateTo

```lua
---@param targetCam Camera
---@param duration number
---@param easeLocation? number
---@param easeRotation? number
cam:InterpolateTo(targetCam, duration, easeLocation, easeRotation)
```

Uses native `SetCamActiveWithInterp` to blend between two cameras.

### Blend

Blends between two sets of camera properties.

```lua

```

### Timeline

Creates a keyframe-based camera animation sequence.

```lua
---@return CameraTimeline
cam:CreateTimeline()
```

```lua
local tl = cam:CreateTimeline()

tl:AddKeyframe({
    coords = vector3(0, 0, 50),
    rotation = vector3(-30, 0, 0),
    fov = 60,
    duration = 2000,
})

tl:AddKeyframe({
    coords = vector3(100, 100, 50),
    rotation = vector3(-10, 0, 90),
    fov = 45,
    duration = 3000,
})

tl:Play(function()
    print('Timeline finished')
end)
```

```lua
tl:Stop()
tl:Reset()
tl:Clear()
```
