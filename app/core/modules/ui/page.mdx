# UI

Client and server-side UI wrapper for `ls_ui`. Access via `LS.UI`.

> Requires `ls_ui` to be started.

## Client

### TextUI

```lua
---@param data TextUI | TextUI[]
LS.UI:TextUI(data)
```

```lua
---@param id string | string[]
LS.UI:HideTextUI(id)
```

```lua
---@param data TextUI
LS.UI:PatchTextUI(data)
```

```lua
LS.UI:HideAllTextUIs()
```

#### TextUI Type

```lua
---@class TextUI
---@field id string
---@field keyText string?
---@field text string
---@field color string?
---@field keyStyle "square" | "hexagon" | "circle"?
---@field position "left-center" | "right-center" | "bottom-center"?
```

### Notification

```lua
---@param data Notification
LS.UI:Notification(data)
```

#### Notification Type

```lua
---@class Notification
---@field title string?
---@field message string?
---@field color string?
---@field icon string?
---@field type "info"|"error"|"warning"|"success"?
---@field duration number?
---@field badges table?
---@field progressType "hexagon"|"circle"?
---@field hideProgress boolean?
---@field position "top-right"|"top-left"|"top-center"|"bottom-right"|"bottom-left"?
```

### InputDialog

Returns `nil` if the user cancels.

```lua
---@param data table
---@return table|nil
LS.UI:InputDialog(data)
```

```lua
LS.UI:CloseInputDialog()
```

### Screen Fade

```lua
---@param duration number
---@param headerMessage? string
LS.UI:ScreenFadeOut(duration, headerMessage)
```

```lua
---@param duration number
LS.UI:ScreenFadeIn(duration)
```

## Server

### Notification

```lua
---@param playerSrc number
---@param data Notification
LS.UI:Notification(playerSrc, data)
```

### Interaction Notification

```lua
---@param playerSrc number
---@param data table
LS.UI:InteractionNotification(playerSrc, data)
```

```lua
---@param playerSrc number
LS.UI:CloseInteractionNotification(playerSrc)
```
