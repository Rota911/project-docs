# Marker

Client-side marker system wrapper for `ls_marker`. Access via `LS.Marker`.

> Requires `ls_marker` to be started.

## Create

```lua
---@param data table
---@return table|nil
LS.Marker:Create(data)
```

```lua
---@param data table
---@return table|nil
LS.Marker:CreateGround(data)
```

```lua
---@param data table
---@return table|nil
LS.Marker:CreateGroundPoly(data)
```

```lua
---@param data table
---@return table|nil
LS.Marker:CreateWaypoint(data)
```

```lua
---@param data table
---@return table|nil
LS.Marker:CreateMark(data)
```
