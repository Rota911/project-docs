# DUI

Client-side DUI (Direct User Interface) wrapper for `ls_marker`. Access via `LS.Dui`.

> Requires `ls_marker` to be started.

## Create

```lua
---@param data DuiProperties
---@return DuiInstance|nil
LS.Dui:Create(data)
```

Creates a new DUI instance rendered in 3D space via `ls_marker`.

## Remove

```lua
---@param dui DuiInstance
LS.Dui:Remove(dui)
```

Destroys the DUI instance and frees its resources.

## Types

```lua
---@class DuiProperties
---@field url string
---@field width number
---@field height number
---@field coords vector3
---@field rotation? vector3
---@field scale? number
---@field [string] any
```

```lua
---@class DuiInstance
---@field handle number
---@field txd number
---@field txn string
---@field [string] any
```

All creation and lifecycle calls proxy to `exports["ls_marker"]`.
