# Proximity

Shared utility functions for finding nearby or closest entities and players. Each function is a direct method on `LS`.

All `maxDistance` parameters default to `2.0`.

## GetNearbyPlayers (Client)

```lua
---@param coords vector3
---@param maxDistance? number
---@param includePlayer? boolean
---@return { id: number, ped: number, coords: vector3 }[]
LS:GetNearbyPlayers(coords, maxDistance, includePlayer)
```

When `includePlayer` is `true`, the local player is included in the results.

## GetNearbyPlayers (Server)

```lua
---@param coords vector3
---@param maxDistance? number
---@return { id: number, ped: number, coords: vector3 }[]
LS:GetNearbyPlayers(coords, maxDistance)
```

## GetClosestPlayer (Client)

```lua
---@param coords vector3
---@param maxDistance? number
---@param includePlayer? boolean
---@return number? playerId
---@return number? playerPed
---@return vector3? playerCoords
LS:GetClosestPlayer(coords, maxDistance, includePlayer)
```

When `includePlayer` is `true`, the local player is considered as a candidate. Returns `nil` for all values if no player is found within `maxDistance`.

## GetRelativeCoords (Shared)

Calculates world coordinates from a position, heading/rotation, and an offset.

```lua
---@overload fun(coords: vector3, heading: number, offset: vector3): vector3
---@overload fun(coords: vector4, offset: vector3): vector4
---@overload fun(coords: vector3, rotation: vector3, offset: vector3): vector3
LS:GetRelativeCoords(coords, rotation, offset)
```

When `rotation` is a `vector3`, full 3D rotation (pitch, roll, yaw) is applied. When it's a `number` (or `coords` is a `vector4`), only yaw rotation is used.

```lua
local front = LS:GetRelativeCoords(GetEntityCoords(ped), GetEntityHeading(ped), vector3(0.0, 2.0, 0.0))
```

## GetFilesInDirectory (Server)

```lua
---@param path string -- prefix with @resourceName/ for other resources
---@param pattern string -- Lua pattern to filter filenames
---@return string[] files
---@return integer fileCount
LS:GetFilesInDirectory(path, pattern)
```

```lua
local files, count = LS:GetFilesInDirectory('data', '%.json$')
local remoteFiles, _ = LS:GetFilesInDirectory('@other_resource/configs', '%.lua$')
```

Uses `dir` on Windows and `ls` on Linux internally.
