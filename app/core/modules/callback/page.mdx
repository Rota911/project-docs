# Callback

Client-server async communication system. Access via `LS.Callback`.

## Client → Server

### LS.Callback()

```lua
---@param event string
---@param delay? number|false
---@param cb function
---@param ... any
LS.Callback(event, delay, cb, ...)
```

### LS.Callback:Await()

Yields the current thread until the server responds.

```lua
---@param event string
---@param delay? number|false
---@param ... any
---@return ...
local result = LS.Callback:Await(event, delay, ...)
```

### LS.Callback:Register()

Registers a callback the server can invoke on this client.

```lua
---@param name string
---@param cb fun(...: any): ...
LS.Callback:Register(name, cb)
```

## Server → Client

### LS.Callback()

```lua
---@param event string
---@param playerId number
---@param cb function
---@param ... any
LS.Callback(event, playerId, cb, ...)
```

### LS.Callback:Await()

Yields the current thread until the client responds.

```lua
---@param event string
---@param playerId number
---@param ... any
---@return ...
local result = LS.Callback:Await(event, playerId, ...)
```

### LS.Callback:Register()

Registers a callback clients can invoke. First param of `cb` is always `source`.

```lua
---@param name string
---@param cb fun(source: number, ...: any): ...
LS.Callback:Register(name, cb)
```

## Example

```lua
-- server
LS.Callback:Register('shop:getPrice', function(source, itemId)
    return Items[itemId]?.price
end)

-- client
local price = LS.Callback:Await('shop:getPrice', false, 'bread')
```

## Timeout

Default callback timeout is `300000ms` (5 min). Override via convar:

```cfg
setr ls:callbackTimeout "300000"
```

The `delay` parameter on client calls prevents repeated calls within the given ms window. Pass `false` to disable.
