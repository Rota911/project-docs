# Server Functions

A Core Server oldalon elérhető főbb funkciói.

## LS:GetPlayer

Lekéri a játékos objektumot azonosító alapján.

**Paraméterek**
- `identifier: number|string` – Player ID, PlayerSrc, Discord ID vagy License2

**Visszatérési érték**
- [`PlayerClass`](/core/Classes/Server/player) – Játékos objektum vagy `nil`

**Támogatott azonosító típusok**
- `source` (1-4 számjegyű szám) – Játékos server source
- `discord` (18 számjegyű szám) – Discord ID
- `license` (string) – License2 azonosító
- `player_id` (>4 számjegyű szám) – Adatbázis player ID

**Példa**
```lua copy
-- Source alapján
local player = LS:GetPlayer(1)

-- Player ID alapján
local player = LS:GetPlayer(10001)

-- Discord alapján
local player = LS:GetPlayer(212704429323976706)

-- License alapján
local player = LS:GetPlayer('98399a05dac2ccae820142f8af6804178b5f178a')

if player then
    print(player.source, player.adminGroup)
end
```

## LS:GetCharacter

Lekéri a karakter objektumot azonosító alapján.

**Paraméterek**
- `identifier: number|string` – Karakter ID vagy PlayerSrc

**Visszatérési érték**
- [`CharacterClass`](/core/Classes/Server/character) – Karakter objektum vagy `nil`

**Megjegyzés**
- 1-4 számjegyű szám: PlayerSrc-ként érzékeli
- Több mint 4 számjegyű szám: CharId-ként érzékeli

**Példa**
```lua copy
-- Source alapán
local character = LS:GetCharacter(1)

-- Karakter ID alapán
local character = LS:GetCharacter(42)

if character then
    print(character.fullName, character.charId)
end
```

## LS:RegisterCommand

Regisztrál egy szerver parancsot.

**Paraméterek**
- commandName: `string|string[]` – Parancs neve vagy nevei
- permission: `string` – Szükséges jogosultság
- cb: `function` – Callback függvény
- params?: `table[]` – Paraméterek tömbje
  - name: `string` – Paraméter neve
  - help?: `string` – Paraméter leírása
  - type?: `'number'|'playerId'|'string'|'longString'` – Paraméter típusa
  - optional?: `boolean` – Opcionális-e
- helpText?: `string` – Parancs leírása

**Visszatérési érték**
- nincs

**Példa**
```lua copy
LS:RegisterCommand({ 'back', 'tpback' }, 'dev', function(source, args, raw)
    local player = args.player or source
    Admin:Back(player)
end, {
    {
        name = 'player',
        help = 'Cél játékos',
        type = 'playerId',
        optional = true
    }
}, 'Visszatérés az előző helyre')
```

## LS:GetCharacters

Lekéri a karaktereket megadott feltételek alapján.

**Paraméterek**
- `key?: string` – Lekérdezés kulcsa (pl. `group`, `playerId`, `charId`)
- `value?: number|string|table` – Lekérdezés értéke vagy értékek táblája

**Visszatérési érték**
- `table` – Karakterek táblája

**Példa 1:** Összes karakter lekérése
```lua copy
local allCharacters = LS:GetCharacters()
for _, character in pairs(allCharacters) do
    print(character.fullName)
end
```

**Példa 2:** Egy adott csoportba tartozó karakterek
```lua copy
local lspdMembers = LS:GetCharacters('group', 'lspd')
print('LSPD tagok száma:', #lspdMembers)
```

**Példa 3:** Egy játékos összes karaktere
```lua copy
local playerCharacters = LS:GetCharacters('playerId', 10001)
```

**Példa 4:** Több csoportba tartozó karakterek
```lua copy
local characters = LS:GetCharacters('group', {'lspd', 'bcso'})
```

## LS:GetGroup

Lekér egy csoport objektumot azonosító alapján.

**Paraméterek**
- `groupId: string` – Csoport azonosító

**Visszatérési érték**
- [`GroupClass`](/core/Classes/Server/group) – Csoport objektum vagy `nil`

**Példa**
```lua copy
local lspd = LS:GetGroup('lspd')
if lspd then
    print(lspd:GetLabel())
    print('Tagok száma:', lspd:GetSize())
end
```

## LS:CreateGroup

Új csoportot hoz létre.

**Paraméterek**
- groupData: `table` – Csoport adatai
  - id: `string` – Csoport azonosító
  - label: `string` – Csoport megjelenítési neve
  - type?: `string` – Csoport típusa (alapértelmezett: `illegal`)
  - limit?: `number` – Maximális létszám (alapértelmezett: 21, -1 = korlátlan)
  - garage?: `string` – Garázs azonosító (alapértelmezett: `legion_square`)

**Visszatérési érték**
- `boolean, string?` – Sikeresség és esetleges hibaüzenet

**Példa**
```lua copy
local success, msg = LS:CreateGroup({
    id = 'lspd',
    label = 'Los Santos Police Department',
    type = 'legal',
    limit = 50,
    garage = 'mrpd'
})

if success then
    print('Csoport sikeresen létrehozva')
else
    print('Hiba:', msg)
end
```

## LS:RemoveGroup

Töröl egy csoportot és eltávolítja az összes tagját.

**Paraméterek**
- `groupName: string` – Csoport azonosító

**Visszatérési érték**
- `boolean` – Sikeres volt-e a törlés

**Megjegyzés**
- Először eltávolítja az összes tagot a csoportból
- Triggeri a `LS:Server:GroupRemoved` eventet

**Példa**
```lua copy
local success = LS:RemoveGroup('lspd')
if success then
    print('Csoport törölve')
end
```

## LS:EditGroup

Szerkeszt egy meglévő csoportot.

**Paraméterek**
- `groupName: string` – Csoport azonosító
- `newData: table` – Frissített csoport adatok
  - `label?: string` – Új megjelenítési név
  - `limit?: number` – Új maximális létszám
  - `garage?: string` – Új garázs azonosító
  - `type?: string` – Új csoport típus

**Visszatérési érték**
- `boolean` – Sikeres volt-e a szerkesztés

**Megjegyzés**
- Csak a megadott mezőket frissíti
- Az adatbázisban is eltárolódnak a változások

**Példa**
```lua copy
local success = LS:EditGroup('lspd', {
    label = 'LSPD - Los Santos Police Department',
    limit = 60
})
```
