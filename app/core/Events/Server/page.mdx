# Server Events

A Core Server oldali eseményei.

## LS:Server:PlayerJoined
Esemény, amikor a játékos sikeresen betölt a szerverre és létrejön a Player objektuma.

**Paraméterek**
- `playerSrc: number` – Játékos forrásazonosító
- `player: PlayerClass` – Játékos objektum

**Megjegyzés**
- Ekkor még nincs bejelentkezve karakterbe
- A Player objektum már elérhető és használható

**Példa**
```lua copy
AddEventHandler('LS:Server:PlayerJoined', function(playerSrc, player)
    print('Játékos csatlakozott:', GetPlayerName(playerSrc))
    print('Admin csoport:', player.adminGroup)
    print('Karakterek száma:', #player.characters)
end)
```

## LS:Server:PlayerLeft
Esemény, amikor a játékos lelép a szerverről.

**Paraméterek**
- `playerSrc: number` – Játékos forrásazonosító
- `reason: string` – Kilépés oka

**Megjegyzés**
- Ez az event a karakter mentése után fut le
- A Player és Character objektumok már törlődtek

**Példa**
```lua copy
AddEventHandler('LS:Server:PlayerLeft', function(playerSrc, reason)
    print(GetPlayerName(playerSrc) .. ' lelépett. Ok:', reason)
end)
```

## LS:Server:CharacterLoaded
Esemény, amikor a karakter sikeresen betöltődik és a Character objektum létrejön.

**Paraméterek**
- `playerSrc: number` – Játékos forrásazonosító
- `charId: number` – Karakter azonosító

**Megjegyzés**
- Az event a `LS:Client:CharacterLoaded` event előtt fut le
- Ekkor már elérhető a Character objektum `LS:GetCharacter()` segítségével

**Példa**
```lua copy
AddEventHandler('LS:Server:CharacterLoaded', function(playerSrc, charId)
    local character = LS:GetCharacter(charId)
    print('Karakter betöltött:', character.fullName)
    print('Csoportok:', json.encode(character.groups))
end)
```

## LS:Server:CharacterLogout
Esemény, amikor a karakter kijelentkezik.

**Paraméterek**
- `playerSrc: number` – Játékos forrásazonosító
- `charId: number` – Karakter azonosító

**Megjegyzés**
- Az event a karakter mentése után fut le
- A Character objektum már törlődött, ne próbáld lekérni
- A Player objektum még elérhető

**Példa**
```lua copy
AddEventHandler('LS:Server:CharacterLogout', function(playerSrc, charId)
    print('Karakter kijelentkezett:', charId)
    local player = LS:GetPlayer(playerSrc)
    if player then
        print('Játékos még online')
    end
end)
```

## LS:Server:SetActiveCharacter
Esemény, amikor a játékos kiválaszt egy karaktert.

**Paraméterek**
- characterId: `number` – Karakter azonosító

**Példa**
```lua copy
RegisterNetEvent('LS:Server:SetActiveCharacter', function(characterId)
    print('Kiválasztott karakter:', characterId)
end)
```

## LS:Server:CharacterCreated
Esemény, amikor a játékos létrehoz egy új karaktert.

**Paraméterek**
- playerSrc: `number` – Játékos forrásazonosító
- characterData: `table` – Új karakter adatai

**Példa**
```lua copy
AddEventHandler('LS:Server:CharacterCreated', function(playerSrc, characterData)
    print(playerSrc, characterData)
end)
```

## LS:Server:CharacterGroupChanged
Esemény, amikor egy karakter csoporttagsága megváltozik.

**Paraméterek**
- playerSrc: `number` – Játékos forrásazonosító
- charId: `number` – Karakter azonosító
- group: `string` – Csoport azonosító
- rank: `string?` – Új rang azonosító (nil = eltávolítva a csoportból)
- groups: `table` – Karakter összes csoportja

**Példa**
```lua copy
AddEventHandler('LS:Server:CharacterGroupChanged', function(playerSrc, charId, group, rank, groups)
    if rank then
        print('Karakter #' .. charId .. ' hozzáadva a(z) ' .. group .. ' csoporthoz')
    else
        print('Karakter #' .. charId .. ' eltávolítva a(z) ' .. group .. ' csoportból')
    end
end)
```

## LS:Server:GroupCreated
Esemény, amikor egy új csoport létrejön.

**Paraméterek**
- groupId: `string` – Csoport azonosító
- groupData: `table` – Csoport adatai

**Példa**
```lua copy
AddEventHandler('LS:Server:GroupCreated', function(groupId, groupData)
    print('Új csoport létrehozva:', groupId, groupData.label)
end)
```

## LS:Server:GroupRemoved
Esemény, amikor egy csoport törlésre kerül.

**Paraméterek**
- groupId: `string` – Törölt csoport azonosító

**Példa**
```lua copy
AddEventHandler('LS:Server:GroupRemoved', function(groupId)
    print('Csoport törölve:', groupId)
end)
```
