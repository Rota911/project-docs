## Core:Server:PlayerJoined
Esemény, amikor a játékos sikeresen betölt a szerverre.

**Paraméterek**
- playerSrc: `number` – Játékos forrásazonosító
- player: `player class` – Játékos objektum

**Példa**
```lua copy
AddEventHandler('Core:Server:PlayerJoined', function(playerSrc, player)
    print(playerSrc, player)
end)
```

## Core:Server:CharacterGroupChanged
Esemény, amikor egy karakter csoporttagsága megváltozik.

**Paraméterek**
- playerSrc: `number` – Játékos forrásazonosító
- charId: `number` – Karakter azonosító
- group: `string` – Csoport azonosító
- rank: `string?` – Új rang azonosító (nil = eltávolítva a csoportból)
- groups: `table` – Karakter összes csoportja

**Példa**
```lua copy
AddEventHandler('Core:Server:CharacterGroupChanged', function(playerSrc, charId, group, rank, groups)
    if rank then
        print('Karakter #' .. charId .. ' hozzáadva a(z) ' .. group .. ' csoporthoz')
    else
        print('Karakter #' .. charId .. ' eltávolítva a(z) ' .. group .. ' csoportból')
    end
end)
```

## Core:Server:GroupCreated
Esemény, amikor egy új csoport létrejön.

**Paraméterek**
- groupId: `string` – Csoport azonosító
- groupData: `table` – Csoport adatai

**Példa**
```lua copy
AddEventHandler('Core:Server:GroupCreated', function(groupId, groupData)
    print('Új csoport létrehozva:', groupId, groupData.label)
end)
```
