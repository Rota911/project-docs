# Client Events

A Core Client oldali eseményei.

## LS:Client:PlayerJoined
Esemény, amikor a játékos sikeresen betölt a szerverre és megkapja a karakterlistát.

**Paraméterek**
- `characters: table` – Karakterek adatai tömbben
- `playerData: table` – Játékos adatai (points, adminGroup, stb.)

**Megjegyzés**
- Ez az event a karakterválasztó megnyitása előtt fut le
- A karakterek listája tartalmazza az összes elérhető karaktert

**Példa**
```lua copy
RegisterNetEvent('LS:Client:PlayerJoined', function(characters, playerData)
    print('Karakterek száma:', #characters)
    print('Admin csoport:', playerData.adminGroup)
    print('Pontok:', playerData.points)
end)
```

## LS:Client:CharacterLoaded
Esemény, amikor a karakter sikeresen betöltődik.

**Paraméterek**
- `characterData: table` – Karakter összes adata
  - `charId: number` – Karakter azonosító
  - `playerId: number` – Játékos azonosító
  - `stateId: number` – Állapot azonosító
  - `firstName: string` – Keresztnév
  - `lastName: string` – Vezetéknév
  - `fullName: string` – Teljes név
  - `dateOfBirth: string` – Születési dátum
  - `nationality: string` – Nemzetiség
  - `gender: string` – Nem
  - `groups: table` – Csoportok
  - `pathType: string` – Életút típusa
  - `position: vector4` – Pozíció
  - `statuses: table` – Státuszok
  - `skills: table` – Skillek

**Példa**
```lua copy
RegisterNetEvent('LS:Client:CharacterLoaded', function(characterData)
    print('Karakter betöltött:', characterData.fullName)
    print('Csoportok:', json.encode(characterData.groups))
    print('Pozíció:', characterData.position)
end)
```

## LS:Client:CharacterDataChanged
Esemény, amikor a karakter adatai megváltoznak.

**Paraméterek**
- `key: string` – Módosított adat kulcsa
- `value: any` – Új érték

**Gyakori kulcsok**
- `position` – Karakter pozíciója változott
- `groups` – Csoporttagság változott
- `statuses` – Státuszok változtak
- `skills` – Skillek változtak

**Példa**
```lua copy
RegisterNetEvent('LS:Client:CharacterDataChanged', function(key, value)
    print('Változott:', key, '- Új érték:', value)
    
    if key == 'groups' then
        print('Csoportok frissültek:', json.encode(value))
    end
end)
```

## LS:Client:PlayerDataChanged
Esemény, amikor a játékos adatai megváltoznak.

**Paraméterek**
- `key: string` – Módosított adat kulcsa
- `value: any` – Új érték

**Gyakori kulcsok**
- `points` – Prémium pontok változtak
- `adminGroup` – Admin csoport változott
- `duty` – Szolgálati állapot változott

**Példa**
```lua copy
RegisterNetEvent('LS:Client:PlayerDataChanged', function(key, value)
    if key == 'points' then
        print('Új pontérték:', value)
    elseif key == 'adminGroup' then
        print('Új admin csoport:', value)
    elseif key == 'duty' then
        print('Szolgálat:', value and 'BE' or 'KI')
    end
end)
```

```lua copy
RegisterNetEvent('LS:Client:CharacterCreated', function(characterData)

end)
```

## LS:Client:CharacterGroupChanged
Esemény, amikor a karakter csoporttagsága megváltozik.

**Paraméterek**
- group: `string` – Csoport azonosító
- rank: `string?` – Új rang azonosító (nil = eltávolítva a csoportból)
- groups: `table` – Karakter összes csoportja

**Példa**
```lua copy
RegisterNetEvent('LS:Client:CharacterGroupChanged', function(group, rank, groups)
    if rank then
        print('Hozzáadva a(z) ' .. group .. ' csoporthoz mint ' .. rank)
    else
        print('Eltávolítva a(z) ' .. group .. ' csoportból')
    end
end)
```

**Paraméterek**
- route: `number` – Routing bucket azonosító

**Példa**

```lua copy
RegisterNetEvent('Routing:Client:NewRoute', function(route)

```
