## Core:Client:PlayerJoined
Esemény, amikor a játékos sikeresen betölt a szerverre.

**Paraméterek**
- characters: `table` – Karakterek adatai

**Példa**
```lua copy
RegisterNetEvent('Core:Client:PlayerJoined', function(characters)
    print(characters)
end)
```

## Core:Client:CharacterGroupChanged
Esemény, amikor a karakter csoporttagsága megváltozik.

**Paraméterek**
- group: `string` – Csoport azonosító
- rank: `string?` – Új rang azonosító (nil = eltávolítva a csoportból)
- groups: `table` – Karakter összes csoportja

**Példa**
```lua copy
RegisterNetEvent('Core:Client:CharacterGroupChanged', function(group, rank, groups)
    if rank then
        print('Hozzáadva a(z) ' .. group .. ' csoporthoz mint ' .. rank)
    else
        print('Eltávolítva a(z) ' .. group .. ' csoportból')
    end
end)
```
