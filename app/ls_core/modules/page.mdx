# Modulok áttekintése

Az LS Core számos beépített modult tartalmaz különböző feladatokhoz. Ez az oldal gyors áttekintést nyújt az elérhető modulokról.

## Kommunikáció

### Callback

Aszinkron kommunikáció szerver és kliens között.

```lua
-- Regisztráció
LS.Callback:Register('getData', function(source, arg)
    return {data = 'value'}
end)

-- Hívás
local result = LS.Callback:Await('getData', false, 'arg')
```

## Utility függvények

### WaitFor

Várakozás egy feltétel teljesüléséig.

```lua
local entity = LS:WaitFor(function()
    return GetEntity()
end, 'Hiba üzenet', 5000)
```

### Table

Kiegészített table műveletek.

```lua
table.deepclone(tbl)
table.merge(t1, t2)
table.contains(tbl, value)
table.shuffle(tbl)
```

### Math

Matematikai segédfüggvények.

```lua
math.clamp(value, min, max)
math.round(value, decimals)
math.groupdigits(1234567) -- "1,234,567"
math.lerp(start, finish, duration)
```

## Streaming / Asset betöltés

### RequestModel

```lua
local modelHash = LS:RequestModel('adder', 10000)
local vehicle = CreateVehicle(modelHash, x, y, z, heading, true, false)
SetModelAsNoLongerNeeded(modelHash)
```

### RequestAnimDict

```lua
LS:RequestAnimDict('anim@heists@box_carry@')
TaskPlayAnim(ped, 'anim@heists@box_carry@', 'idle', 8.0, -8.0, -1, 49, 0)
```

### RequestAnimSet

```lua
LS:RequestAnimSet('move_m@drunk@slightlydrunk')
SetPedMovementClipset(ped, 'move_m@drunk@slightlydrunk', 1.0)
```

### RequestStreamedTextureDict

```lua
LS:RequestStreamedTextureDict('shopui_title_gr_gunmod', 5000)
```

### RequestScaleformMovie

```lua
local scaleform = LS:RequestScaleformMovie('INSTRUCTIONAL_BUTTONS')
```

### RequestWeaponAsset

```lua
local weaponHash = GetHashKey('WEAPON_PISTOL')
LS:RequestWeaponAsset(weaponHash, 10000)
```

## Zones

Terület kezelés különböző formákban.

### Sphere Zone

```lua
LS.Zones:Sphere({
    coords = vector3(100, 200, 30),
    radius = 50.0,
    onEnter = function(self) print('Belépés') end,
    onExit = function(self) print('Kilépés') end
})
```

### Box Zone

```lua
LS.Zones:Box({
    coords = vector3(100, 200, 30),
    size = vector3(10, 10, 5),
    rotation = 45,
    onEnter = function(self) print('Belépés') end
})
```

### Poly Zone

```lua
LS.Zones:Poly({
    points = {
        vector3(100, 200, 30),
        vector3(150, 200, 30),
        vector3(150, 250, 30),
        vector3(100, 250, 30)
    },
    thickness = 10,
    onEnter = function(self) print('Belépés') end
})
```

## Használati példák

### Teljes példa: Játékos interakció

```lua
-- Legközelebbi játékos keresése
local playerId, ped, coords = LS:GetClosestPlayer(GetEntityCoords(cache.ped), 5.0)

if playerId then
    -- Adat kérése a szervertől
    local playerData = LS.Callback:Await('getPlayerData', false, playerId)
    
    -- Feldolgozás
    print('Játékos neve:', playerData.name)
end
```

### Teljes példa: Jármű spawn

```lua
CreateThread(function()
    -- Model betöltése
    local modelHash = LS:RequestModel('adder')
    
    -- Animáció betöltése
    LS:RequestAnimDict('anim@mp_player_intmenu@key_fob@')
    
    -- Jármű létrehozása
    local ped = cache.ped
    local coords = GetEntityCoords(ped)
    local heading = GetEntityHeading(ped)
    
    local vehicle = CreateVehicle(modelHash, coords.x, coords.y, coords.z, heading, true, false)
    
    -- Animáció lejátszása
    TaskPlayAnim(ped, 'anim@mp_player_intmenu@key_fob@', 'fob_click', 8.0, -8.0, -1, 48, 0)
    
    -- Cleanup
    SetModelAsNoLongerNeeded(modelHash)
    
    -- Játékos beültetése
    LS:WaitFor(function()
        if DoesEntityExist(vehicle) then
            return true
        end
    end, 'Jármű létrehozás sikertelen', 3000)
    
    TaskWarpPedIntoVehicle(ped, vehicle, -1)
end)
```

### Teljes példa: Zone alapú shop

```lua
local shopZone = LS.Zones:Sphere({
    coords = vector3(25.7, -1347.3, 29.5),
    radius = 3.0,
    onEnter = function(self)
        lib.notify({
            title = 'Üzlet',
            description = 'Nyomd meg az E gombot',
            type = 'info'

    end,
    onExit = function(self)
        self.insideShop = false
    end,
    inside = function(self)
        if IsControlJustPressed(0, 38) then -- E
            TriggerEvent('shop:open')
        end
    end
})
```

## Modulok kombinálása

Az LS Core moduljai remekül együttműködnek:

```lua
-- Model betöltés + WaitFor + NearbyVehicles
CreateThread(function()
    local modelHash = LS:RequestModel('adder')
    
    local vehicle = CreateVehicle(modelHash, x, y, z, heading, true, false)
    
    LS:WaitFor(function()
        return DoesEntityExist(vehicle)
    end, 'Jármű létrehozás hiba', 5000)
    
    SetModelAsNoLongerNeeded(modelHash)
    
    -- Közeli járművek listázása
    Wait(1000)
    local coords = GetEntityCoords(vehicle)
    local nearby = LS:GetNearbyVehicles(coords, 50.0)
    
    print('Közeli járművek száma:', #nearby)
end)
```

## Modulok teljesítménye

Az LS Core moduljai optimalizáltak:

- **Grid rendszer** - Spatial partitioning a gyors entity kereséshez
- **Cache rendszer** - Csökkenti az ismételt számításokat
- **Lazy loading** - Modulok csak akkor töltődnek be, amikor használod őket
- **Automatikus cleanup** - Resource leállításkor minden megtisztul
