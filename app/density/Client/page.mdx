# Client Funkciók

A client oldali rendszer felelős a populációs szorzók alkalmazásáért és a zóna-alapú beállítások kezeléséért.

## Funkciók

### `applyDensityMultipliers(data)`

Alkalmazza a populációs szorzókat az aktuális frame-ben.

**Paraméterek:**
- `data` (table): Populációs beállítások objektum
  - `peds` (number): Gyalogos sűrűség (0.0 - 1.0)
  - `vehicles` (number): Jármű sűrűség (0.0 - 1.0)
  - `randomVehicles` (number): Véletlenszerű járművek sűrűsége (0.0 - 1.0)
  - `parked` (number): Parkolt járművek sűrűsége (0.0 - 1.0)
  - `scenario` (number): Scenario sűrűség (0.0 - 1.0)

**Művelet:**
- Beállítja a parkolt járművek sűrűségét
- Beállítja a járművek sűrűségét
- Beállítja a véletlenszerű járművek sűrűségét
- Beállítja a gyalogosok sűrűségét
- Beállítja a scenario sűrűségét
- Letiltja a jármű jutalmakat
- Letiltja a távoli rendőrautó szirénákat

---

### `getScheduledPopulation(schedules)`

Megkeresi a megfelelő populációs beállítást az aktuális időpont alapján.

**Paraméterek:**
- `schedules` (table[]): Ütemezések tömbje, ahol minden elem tartalmazza:
  - `from` (number): Kezdő óra (0-24)
  - `to` (number): Befejező óra (0-24)
  - `data` (table): Populációs beállítások

**Visszatérési érték:**
- `table`: Az időponthoz tartozó populációs beállítások, vagy üres tábla

**Példa:**
```lua
local schedules = {
    {
        from = 0,
        to = 6,
        data = {
            peds = 0.2,
            vehicles = 0.3,
            randomVehicles = 0.2,
            parked = 0.8,
            scenario = 0.1
        }
    },
    {
        from = 6,
        to = 12,
        data = {
            peds = 0.8,
            vehicles = 0.9,
            randomVehicles = 0.7,
            parked = 0.3,
            scenario = 0.8
        }
    }
}

local currentSettings = getScheduledPopulation(schedules)
```

### `getCurrentZone()`

Meghatározza a játékos aktuális zónáját és visszaadja a hozzá tartozó populációs beállításokat.

**Visszatérési érték:**
- `table`: Populációs beállítások objektum a következő mezőkkel:
  - `peds` (number)
  - `vehicles` (number)
  - `randomVehicles` (number)
  - `parked` (number)
  - `scenario` (number)

**Működés:**
1. Ellenőrzi a `GlobalState.Population` értékét (admin felülbírálás)
2. Ha nincs globális beállítás, lekéri a játékos koordinátáit
3. Meghatározza a zónát a koordináták alapján
4. Ha van specifikus zóna konfiguráció, azt használja
5. Egyébként az alapértelmezett beállításokat alkalmazza

## Thread-ek

### Zone Thread

**Frissítési gyakoriság:** 2000ms (2 másodperc)

**Működés:**
- Folyamatosan figyelemmel kíséri a játékos pozícióját
- Frissíti a `currentData` változót az aktuális zóna beállításaival
- Csak akkor fut, ha a játékos be van jelentkezve (`LocalPlayer.state.isLoggedIn`)

```lua
local function zoneThread()
    while LocalPlayer.state.isLoggedIn do
        currentData = getCurrentZone()
        Wait(2000)
    end
end
```

---

### Density Thread

**Frissítési gyakoriság:** Minden frame (0ms)

**Működés:**
- Folyamatosan alkalmazza a populációs szorzókat
- A `currentData` változó értékeit használja
- Csak akkor fut, ha a játékos be van jelentkezve

```lua
local function densityThread()
    while LocalPlayer.state.isLoggedIn do
        applyDensityMultipliers(currentData)
        Wait(0)
    end
end
```

## Események

### `Core:Client:CharacterLoaded`

Akkor aktiválódik, amikor egy karakter betöltődik.

**Művelet:**
1. Elindítja a Zone Thread-et
2. 1 másodperc várakozás
3. Elindítja a Density Thread-et

```lua
RegisterNetEvent('Core:Client:CharacterLoaded', function()
    CreateThread(zoneThread)
    Wait(1000)
    CreateThread(densityThread)
end)
```

---

### `onResourceStart`

Akkor aktiválódik, amikor az erőforrás elindul.

**Művelet:**
- Ellenőrzi, hogy a játékos be van-e jelentkezve
- Ha igen, elindítja mindkét thread-et

```lua
AddEventHandler('onResourceStart', function(resourceName)
    if resourceName ~= GetCurrentResourceName() then
        return
    end

    if LocalPlayer.state.isLoggedIn then
        CreateThread(zoneThread)
        Wait(1000)
        CreateThread(densityThread)
    end
end)
```

## Inicializálás

Az erőforrás indításakor a következők történnek:

### 1. Kapcsolatok Beállítása

Beállítja az NPC csoportok közötti kapcsolatokat a konfiguráció alapján:

```lua
for _, rel in ipairs(Config.Relationships) do
    SetRelationshipBetweenGroups(rel.relationship, rel.group1, rel.group2)
end
```

**Kapcsolat típusok:**
- `0` = Companion (Társ)
- `1` = Respect (Tisztelet)
- `2` = Like (Tetszik)
- `3` = Neutral (Semleges)
- `4` = Dislike (Nem tetszik)
- `5` = Hate (Gyűlölet)
- `255` = Pedestrians (Gyalogosok)

---

### 2. Scenario-k Letiltása

Letiltja a nem kívánt scenario-kat:

```lua
for i = 1, #Config.Scenarios do
    SetScenarioTypeEnabled(Config.Scenarios[i], false)
end
```

---

### 3. Rendőrség Letiltása

Letiltja a játékbeli rendőrségi funkciókat:

```lua
SetMaxWantedLevel(0)
SetCreateRandomCopsNotOnScenarios(false)
SetCreateRandomCops(false)
SetCreateRandomCopsOnScenarios(false)

for i = 1, 32 do
    EnableDispatchService(i, false)
end
```

---

### 4. Blokolt Területek

Hozzáadja a scenario blokkoló területeket és népesség csökkentő területeket:

```lua
for i = 1, #Config.ScenarioBlockedAreas do
    local area = Config.ScenarioBlockedAreas[i]
    local centerPoint = area.centerPoint
    local radiusSize = area.radius or 180.0

    AddScenarioBlockingArea(centerPoint - radiusSize, centerPoint + radiusSize, false, true, true, true)
    AddPopMultiplierArea(centerPoint - radiusSize, centerPoint + radiusSize, 0.0, 0.0, false)
    SetPedNonCreationArea(centerPoint - radiusSize, centerPoint + radiusSize)
end
```

**Speciális terület:**
- Repülőgép spawn letiltó terület: `-3966.93, -3934.72, 200` és `1429.366, -224.4396, 2000` koordináták között

## Optimalizálás

A rendszer optimalizált teljesítményre:

- A natív függvények lokális változókban vannak tárolva a gyorsabb hozzáférés érdekében
- A Zone Thread csak 2 másodpercenként frissít (nem szükséges gyakoribb ellenőrzés)
- A Density Thread minden frame-ben fut, de csak egyszerű natív hívásokat használ

```lua
local setParkedVehicleDensityMultiplierThisFrame = SetParkedVehicleDensityMultiplierThisFrame
local setVehicleDensityMultiplierThisFrame = SetVehicleDensityMultiplierThisFrame
local setRandomVehicleDensityMultiplierThisFrame = SetRandomVehicleDensityMultiplierThisFrame
local setPedDensityMultiplierThisFrame = SetPedDensityMultiplierThisFrame
local setScenarioPedDensityMultiplierThisFrame = SetScenarioPedDensityMultiplierThisFrame
```
