# Density System

A **Density System** (ls_density) egy átfogó populáció-kezelő rendszer, amely dinamikusan szabályozza a gyalogosok, járművek és scenario-k sűrűségét a szervereden.

## Főbb Funkciók

- **Dinamikus populáció szabályozás**: Zónánként és időbeosztás szerint változtatható a gyalogosok és járművek száma
- **Időalapú ütemezés**: Különböző populációs beállítások különböző napszakokra
- **Globális felülbírálás**: Admin parancsokkal a teljes szerver populációja állítható
- **Scenario letiltás**: Nem kívánt scenario-k kikapcsolása
- **Kapcsolatok kezelése**: NPC csoportok közötti kapcsolatok beállítása
- **Blokolt területek**: Meghatározott területeken a populáció teljes letiltása

## Működési Elv

A rendszer két fő szálon működik:

1. **Zone Thread** (2 másodpercenként): Ellenőrzi a játékos aktuális zónaját és betölti a megfelelő beállításokat
2. **Density Thread** (minden frame-ben): Alkalmazza a betöltött populációs szorzókat

## Installálás

1. Helyezd az erőforrást a `resources/[ls]/ls_density` mappába
2. Add hozzá a `server.cfg`-hez:
   ```bash
   ensure ls_density
   ```
3. Az erőforrás automatikusan betöltődik a karakter bejelentkezésekor

## Függőségek

- `ls_core`: Core rendszer integrációhoz

## Alapértelmezett Beállítások

Az alapértelmezett populációs beállítások időbeosztás szerint:

| Időszak | Gyalogosok | Járművek | Véletlen Járművek | Parkolt | Scenario |
|---------|-----------|----------|-------------------|---------|----------|
| 0-5     | 0.3       | 0.2      | 0.1              | 0.8     | 0.1      |
| 5-9     | 0.8       | 0.9      | 0.8              | 0.1     | 0.9      |
| 9-16    | 0.3       | 0.2      | 0.1              | 0.8     | 0.1      |
| 16-20   | 0.8       | 0.9      | 0.8              | 0.1     | 0.9      |
| 20-24   | 0.8       | 0.4      | 0.5              | 0.6     | 0.9      |

*Az értékek 0.0 és 1.0 között mozognak, ahol 1.0 a maximális sűrűség.*

## Zónák

A rendszer támogatja egyedi beállítások megadását különböző zónákra. Alapértelmezetten csak az ALTA zóna van konfigurálva 0.5-ös értékekkel minden paraméterhez.

## Admin Parancsok

### `/setpop`

Globálisan beállítja a populációt az egész szerveren.

```
/setpop [peds] [vehicles] [randomVehicles] [parked] [scenario]
```

**Példa:**
```
/setpop 0.5 0.6 0.4 0.7 0.5
```

### `/resetpop`

Visszaállítja a populációt az alapértelmezett (zóna és időalapú) beállításokra.

```
/resetpop
```
