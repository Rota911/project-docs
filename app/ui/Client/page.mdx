## UI:Notification()
Értesítést jelenít meg a képernyőn.

**Paraméterek**
- title?: `string` – Az értesítés címe
- message?: `string` – Fő üzenet vagy tartalom
- color?: `string` – Színtéma (Mantine UI szín)
- icon?: `string` – Megjelenítendő ikon
- type?: `"info"|"error"|"warning"|"success"` – Értesítés típusa
- duration?: `number` – Időtartam ms-ban automatikus bezárásig
- badges?: `table` – Badge-ek tömbje (string vagy BadgeItem)
- startTime?: `number` – Értesítés indításának időbélyege
- progressType?: `"hexagon"|"circle"` – Folyamatjelző alakja
- hideProgress?: `boolean` – Ha igaz, elrejti a folyamatjelzőt
- position?: `"top-right"|"top-left"|"top-center"|"bottom-right"|"bottom-left"` – Képernyő pozíció

1. példa: Egyszerű értesítés
```lua copy
UI:Notification({
    title = 'Siker',
    message = 'A művelet sikeresen befejeződött!',
    type = 'success'
})
```

2. példa: Részletes értesítés beállításokkal
```lua copy
UI:Notification({
    title = 'Figyelmeztetés',
    message = 'Ez egy fontos üzenet',
    type = 'warning',
    duration = 10000,
    progressType = 'hexagon',
    position = 'top-right',
    color = 'orange',
    icon = 'AlertTriangle'
})
```

3. példa: Hiba értesítés folyamatjelző nélkül
```lua copy
UI:Notification({
    title = 'Hiba',
    message = 'Valami rosszul ment',
    type = 'error',
    hideProgress = true,
    position = 'top-center'
})
```

## UI:TextUI()
Megjeleníti a TextUI-t a képernyőn.

**Paraméterek**
- data: `table|table[]` – TextUI vagy TextUI-k tömbje
  - id: `string` – Egyedi azonosító
  - text: `string` – Megjelenítendő szöveg
  - keyText?: `string` – Gomb szövege
  - icon?: `string` – Ikon neve
  - color?: `string` – Szín
  - keyStyle?: `"square"|"hexagon"|"circle"` – Gomb stílusa
  - position?: `"left-center"|"right-center"|"bottom-center"` – Pozíció a képernyőn

1. példa: Egyszerű TextUI megjelenítése
```lua copy
UI:TextUI({
    id = 'ls_digout',
    text = 'Kiásás',
    keyText = 'E'
})
```

2. példa: TextUI színnel és stílussal
```lua copy
UI:TextUI({
    id = 'vehicle_lock',
    text = 'Jármű zárása/nyitása',
    keyText = 'L',
    color = 'blue',
    keyStyle = 'hexagon',
    position = 'left-center'
})
```

3. példa: Több TextUI egyszerre
```lua copy
UI:TextUI({
    {
        id = 'prop_place',
        text = 'Elhelyezés',
        keyText = 'E',
        position = 'bottom-center'
    },
    {
        id = 'prop_rotate',
        text = 'Forgatás',
        keyText = 'R',
        position = 'bottom-center'
    },
    {
        id = 'prop_cancel',
        text = 'Mégsem',
        keyText = 'X',
        color = 'red',
        position = 'bottom-center'
    }
})
```

## UI:HideTextUI()
Elrejti a TextUI-t.

**Paraméterek**
- id: `string`|`table` – Azonosító vagy azonosítók tömbje

1. példa: Egy TextUI elrejtése
```lua copy
UI:HideTextUI('ls_digout')
```

2. példa: Több TextUI elrejtése egyszerre
```lua copy
UI:HideTextUI({
    'props_place',
    'props_cancel',
    'props_rotate_right',
    'props_rotate_left',
    'props_quick_rotate'
})
```

## UI:Progressbar()
Progressbar-t jelenít meg és visszaadja az eredményt.

**Paraméterek**
- label?: `string` – Megjelenített felirat
- icon?: `string` – Ikon neve
- color?: `string` – Szín
- duration: `number` – Időtartam ms-ban
- timerType?: `"seconds"|"percentage"` – Időzítő megjelenítési típusa
- progressType?: `"hexagon"|"circle"` – Progressbar alakja
- useWhileDead?: `boolean` – Használható-e halott állapotban
- canCancel?: `boolean` – Megszakítható-e (X gomb)
- disable?: `table` – Letiltandó akciók
  - mouse?: `boolean` – Egér mozgatás letiltása
  - move?: `boolean` – Karakter mozgatás letiltása
  - sprint?: `boolean` – Futás letiltása
  - car?: `boolean` – Járművezetés letiltása
  - combat?: `boolean` – Harc letiltása
- anim?: `table` – Animáció beállítások
  - dict?: `string` – Animáció szótár neve
  - clip?: `string` – Animáció klip neve
  - scenario?: `string` – Scenario neve (dict helyett)
  - blendIn?: `number` – Blend in érték (alapértelmezett: 3.0)
  - blendOut?: `number` – Blend out érték (alapértelmezett: 1.0)
  - duration?: `number` – Animáció időtartama (alapértelmezett: -1)
  - flag?: `number` – Animáció flag (alapértelmezett: 49)
  - playbackRate?: `number` – Lejátszási sebesség (alapértelmezett: 0)
  - lockX?: `boolean` – X tengely zárolása
  - lockY?: `boolean` – Y tengely zárolása
  - lockZ?: `boolean` – Z tengely zárolása
  - playEnter?: `boolean` – Scenario belépési animáció lejátszása (alapértelmezett: true)
- prop?: `table|table[]` – Prop vagy propok tömbje
  - model: `string|number` – Prop model neve vagy hash-e
  - bone?: `number` – Csont azonosító (alapértelmezett: 60309)
  - pos: `vector3` – Pozíció eltolás
  - rot: `vector3` – Forgatás
  - rotOrder?: `number` – Forgatási sorrend (alapértelmezett: 0)

**Visszatérési érték**
- `boolean` – `true` ha sikeresen befejeződött, `false` ha megszakadt

1. példa: Egyszerű progressbar animációval
```lua copy
local success = UI:Progressbar({
    duration = 5000,
    label = 'Tankolás',
    icon = 'GasStation',
    color = 'blue',
    useWhileDead = false,
    canCancel = true,
    disable = {
        move = true,
        combat = true,
    },
    anim = {
        dict = 'timetable@gardener@filling_can',
        clip = 'gar_ig_5_filling_can',
    },
})

if success then
    print('Tankolás sikeres')
else
    print('Tankolás megszakítva')
end
```

2. példa: Progressbar scenario animációval
```lua copy
if UI:Progressbar({
    duration = 3000,
    label = 'Javítás',
    icon = 'Wrench',
    color = 'orange',
    canCancel = true,
    disable = {
        move = true,
        car = true,
        combat = true,
    },
    anim = {
        scenario = 'WORLD_HUMAN_WELDING',
    },
}) then
    print('Javítás kész')
end
```

3. példa: Progressbar prop-pal
```lua copy
local success = UI:Progressbar({
    duration = 4000,
    label = 'Telefon használata',
    icon = 'Phone',
    canCancel = true,
    disable = {
        move = true,
        sprint = true,
        combat = true,
    },
    anim = {
        dict = 'cellphone@',
        clip = 'cellphone_text_read_base',
        flag = 49,
    },
    prop = {
        model = 'prop_phone_01',
        bone = 28422,
        pos = vector3(0.0, 0.0, 0.0),
        rot = vector3(0.0, 0.0, 0.0),
    },
})
```

4. példa: Progressbar több prop-pal
```lua copy

        {
            model = 'prop_tool_wrench',

            model = 'prop_tool_screwdvr01',

            pos = vector3(0.0, 0.0, 0.0),
```

## UI:CancelProgressbar()
Megszakítja az aktív progressbar-t.

```lua copy
if not LocalPlayer.state.isLoggedIn then
    UI:CancelProgressbar()
end
```

## UI:IsProgressbarActive()
Visszaadja, hogy van-e aktív progressbar.

**Visszatérési érték**
- `boolean` – `true` ha van aktív progressbar, `false` ha nincs

```lua copy
local isActive = UI:IsProgressbarActive()
```
