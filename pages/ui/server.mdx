## UI:Notification()
Értesítést jelenít meg a képernyőn.

**Paraméterek**
- title?: `string`
- message?: `string`
- color?: `string`
- icon?: `string`
- type?: `"info"|"error"|"warning"|"success"`
- duration?: `number`
- badges?: `table`
- startTime?: `number`
- progressType?: `"hexagon"|"circle"`
- hideProgress?: `boolean`
- position?: `"top-right"|"top-left"|"top-center"|"bottom-right"|"bottom-left"`

```lua copy
UI:Notification({
    title = 'Cím',
    message = 'Üzenet',
    duration = 10000,
    progressType = 'hexagon',
    position = 'top-right'
})
```
