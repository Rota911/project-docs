## Core:Server:PlayerJoined
Esemény, amikor a játékos sikeresen betölt a szerverre.

**Paraméterek**
- playerSrc: `number` – Játékos forrásazonosító
- player: `player class` – Játékos objektum

**Példa**
```lua copy
AddEventHandler('Core:Server:PlayerJoined', function(playerSrc, player)
    print(playerSrc, player)
end)
```

## Core:Server:SetActiveCharacter
Esemény, amikor a játékos kiválaszt egy karaktert.

**Paraméterek**
- characterId: `number` – Karakter azonosító

**Példa**
```lua copy
RegisterNetEvent('Core:Server:SetActiveCharacter', function(characterId)
    print('Kiválasztott karakter:', characterId)
end)
```
