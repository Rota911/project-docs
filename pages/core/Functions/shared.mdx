# Shared functions

## LS:RegisterModule
Regisztrál egy új komponenst vagy modult a rendszerben.

**Paraméterek**
- componentName: `string` – Komponens neve
- componentData: `table` – Komponens adatai

**Visszatérési érték**
- `boolean` – Sikeres regisztráció

**Példa**
```lua copy
local MyComponent = { _name = 'MyComponent', _required = { 'foo' } }
LS:RegisterModule('MyComponent', MyComponent)
```
## LS:ImportModule
Lekéri egy regisztrált komponens vagy modul adatait.

**Paraméterek**
- componentName: `string` – Komponens neve

**Visszatérési érték**
- `table|nil` – Komponens adatai vagy nil, ha nem létezik

**Példa**
```lua copy
local MyComponent = LS:ImportModule('MyComponent')
if MyComponent then
	print(MyComponent._name)
end
```
## LS:ImportDependencies
Betölti a komponens függőségeit és meghívja a callbacket, ha minden dependency elérhető.

**Paraméterek**
- componentData: `string` – A komponens neve, amelynek függőségeit importáljuk
- dependencies: `table` – Függőségek neveinek tömbje
- cb: `function` – Callback függvény, amit akkor hív, ha minden dependency elérhető

**Visszatérési érték**
- nincs

**Példa**
```lua copy
LS:ImportDependencies('MyComponent', { 'OtherComponent', 'Database' }, function(errors)
	if #errors == 0 then
		print('Minden dependency betöltve!')
	else
		print('Hibás dependency:', table.concat(errors, ', '))
	end
end)
```
## LS:ExtendComponent
Kiterjeszti egy már regisztrált komponenst új adatokkal.

**Paraméterek**
- componentName: `string` – Komponens neve
- componentData: `table` – Új adatok, amivel kiegészítjük a komponenst

**Visszatérési érték**
- nincs

**Példa**
```lua copy
LS:ExtendComponent('MyComponent', { foo = 'bar', version = 2 })
```
